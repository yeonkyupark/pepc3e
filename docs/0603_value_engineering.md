## 가치공학(Value Engineering)

가치공학(Value Engineering, VE)은 제품이나 서비스가 수행해야 할 기능(Function)을 체계적으로 분석하여 필요한 기능은 유지하거나 향상시키면서 총비용(Cost)을 최소화함으로써 가치를 극대화하는 관리기법이다. 제2차 세계대전 당시 대체자재 검토 과정에서 시작되었으며, 현재는 제품개발, 생산설계, 원가절감, 공정개선, 구매전략 등 전 분야에서 활용되는 대표적인 가치혁신 기법으로 발전하였다.

```text
가치(Value)
=
기능(Function)
────────────
비용(Cost)
```

가치공학의 궁극적인 목적은 단순한 비용절감이 아니라 고객이 요구하는 기능을 가장 경제적으로 달성하는 데 있다.

## 가치공학의 개요

### 가치공학의 정의

기능 중심 사고(Function-Oriented Thinking)를 기반으로 제품·공정·서비스의 기능을 분석하고 불필요한 비용을 제거하여 가치를 향상시키는 체계적 방법론이다.

### 가치공학의 목적

* 고객가치 향상
* 원가절감
* 기능개선
* 품질향상
* 수익성 향상
* 경쟁력 확보

### 가치공학의 기본원리

#### 기능 중심 사고

가치공학은 제품 자체가 아니라 제품이 수행하는 기능을 대상으로 분석한다.

예를 들어 볼펜의 기능은 "글씨를 쓴다", 의자의 기능은 "사람을 지지한다"로 표현된다.

#### 가치 향상 방법

```text
기능 향상 + 비용 유지
기능 유지 + 비용 절감
기능 향상 + 비용 절감
```

가치공학은 위 세 가지 방향을 통해 가치를 향상시킨다.

## 가치공학의 발전과정

### 역사적 배경

| 시기     | 주요 내용                       |
| ------ | --------------------------- |
| 1940년대 | 대체자재 검토 활동                  |
| 1947년  | Lawrence D. Miles의 VE 체계 정립 |
| 1960년대 | 미국 국방부 및 공공사업 적용            |
| 1980년대 | 제조업 중심 확산                   |
| 현재     | 제품개발·서비스·건설·공공사업 확대         |

### VE와 VA

#### VE(Value Engineering)

설계 및 개발단계에서 수행하는 가치향상 활동

#### VA(Value Analysis)

기존 제품을 대상으로 수행하는 가치향상 활동

| 구분   | VE      | VA      |
| ---- | ------- | ------- |
| 적용시점 | 개발 전    | 생산 후    |
| 목적   | 사전 원가절감 | 사후 원가개선 |
| 효과   | 예방적 접근  | 개선적 접근  |

## 기능분석(Function Analysis)

### 기능의 정의

기능(Function)은 고객이 제품이나 서비스에 기대하는 역할 또는 목적이다.

### 기능의 표현

기능은 일반적으로 동사 + 명사 형태로 표현한다.

| 대상  | 기능       |
| --- | -------- |
| 볼트  | 부품을 고정한다 |
| 모터  | 동력을 전달한다 |
| 포장재 | 제품을 보호한다 |

### 기능의 분류

#### 기본기능(Basic Function)

제품 존재의 핵심 목적

#### 부가기능(Secondary Function)

기본기능을 지원하거나 고객 편의성을 제공하는 기능

| 구분   | 내용             |
| ---- | -------------- |
| 기본기능 | 제품의 존재 이유      |
| 부가기능 | 품질, 디자인, 편의성 등 |

### 기능가치 분석

```text
기능
 ↓
원가 측정
 ↓
가치 평가
 ↓
개선 대상 선정
```

기능 대비 비용이 과도한 요소가 개선 대상이 된다.

### 실행단계

## FAST(Function Analysis System Technique)

### 개요

기능 간 논리적 관계를 분석하는 대표적인 기능분석 기법이다.

### 기본원리

#### How-Why Logic

```text
왜(Why)?
←────────

기능 A ──→ 기능 B

────────→
어떻게(How)?
```

### FAST 활용 목적

* 기능 구조 파악
* 불필요 기능 제거
* 개선 아이디어 도출

### FAST의 장점

* 기능 중심 분석 가능
* 개선 대상 명확화
* 팀 의사소통 향상

## 가치공학의 주요 기법

### 기능원가 분석

기능별 원가를 분석하여 개선 대상을 선정한다.

### Pareto 분석

중요 기능과 원가 집중 영역을 파악한다.

### 브레인스토밍

창의적 대안을 도출한다.

### 벤치마킹

우수 사례를 비교 분석한다.

### TRIZ

발명의 원리를 활용하여 혁신적 대안을 도출한다.

### QFD 연계

고객 요구사항을 기능 요구사항으로 변환한다.

```text
VOC
 ↓
QFD
 ↓
기능 정의
 ↓
VE 수행
```

## 원가절감과 가치공학

### 원가절감과의 차이

| 구분 | 원가절감  | 가치공학     |
| -- | ----- | -------- |
| 목적 | 비용 감소 | 가치 향상    |
| 접근 | 비용 중심 | 기능 중심    |
| 품질 | 저하 가능 | 유지 또는 향상 |
| 관점 | 단기적   | 장기적      |

### 목표원가관리와의 관계

```text
시장가격
 ↓
목표원가 설정
 ↓
VE 수행
 ↓
목표원가 달성
```

목표원가관리(Target Costing)의 핵심 실행수단이 가치공학이다.

## 제품개발과 가치공학

### 동시공학(Concurrent Engineering)

설계·생산·구매·품질 부문이 동시에 참여하여 가치를 향상시킨다.

### DFX(Design for X)

설계단계에서 다양한 관점을 반영한다.

#### 주요 유형

* DFM(생산 용이성)
* DFA(조립 용이성)
* DFS(서비스 용이성)
* DFR(신뢰성 향상)
* DFE(환경친화 설계)

### VE와 DFX 관계

DFX가 설계 원칙이라면 VE는 가치 향상을 위한 체계적 추진방법론이다.

## 가치공학과 생산혁신

### Lean 생산과 VE

| Lean  | VE     |
| ----- | ------ |
| 낭비 제거 | 기능 최적화 |
| 공정 중심 | 기능 중심  |
| 운영 개선 | 설계 개선  |

Lean은 운영단계, VE는 설계단계에서 가치 향상을 추구한다.

### TPS와 VE

TPS가 생산과정의 낭비를 제거한다면 VE는 설계단계에서 불필요한 기능원가를 제거한다.

### Six Sigma와 VE

```text
VE
→ 가치 향상

Six Sigma
→ 변동 감소

Lean
→ 낭비 제거
```

세 기법은 상호 보완적으로 활용된다.

## 가치공학의 한계

### 기능 정량화의 어려움

고객가치를 수치로 표현하기 어렵다.

### 창의성 의존

개선성과가 팀 역량에 영향을 받는다.

### 초기 투자 필요

분석과 검토에 상당한 시간과 비용이 소요된다.

### 단기성과 한계

즉각적인 재무성과가 나타나지 않을 수 있다.

## 가치공학의 최근 동향

### Target Value Design

목표원가와 목표가치를 동시에 관리

### Design-to-Cost

설계단계 원가최적화 강화

### Sustainable VE

ESG와 탄소중립을 고려한 가치혁신

### AI 기반 VE

생성형 AI와 최적화 알고리즘 활용 확대

### Smart Engineering

PLM, Digital Twin, AI를 연계한 가치공학 체계 구축

## 가치공학의 종합

가치공학은 기능 중심 사고를 기반으로 고객이 요구하는 기능은 유지 또는 향상시키면서 비용을 최소화하여 가치를 극대화하는 체계적 혁신기법이다. 전통적인 원가절감 활동이 비용 자체를 줄이는 데 초점을 두었다면, 가치공학은 기능과 비용의 균형을 최적화하여 고객가치와 기업수익성을 동시에 향상시키는 데 목적이 있다. 최근에는 목표원가관리, DFX, Lean, Digital Twin, AI 기술과 융합되면서 설계단계 중심의 전략적 가치혁신 도구로 발전하고 있다.

### 범위 연계

| 연계 영역  | 주요 내용                |
| ------ | -------------------- |
| 원가관리   | 목표원가관리, 원가기획         |
| 제품개발기술 | 동시공학, DFX, PLM       |
| 품질관리   | QFD, 고객가치            |
| 생산성관리  | 가치향상과 생산성            |
| 공정개선   | Lean, TPS, Six Sigma |

| 스마트팩토리 | Digital Twin, AI     |
| 생산혁신기술 | TRIZ, 혁신설계           |

### 각주

[^1]: 가치공학은 미국의 GE(General Electric)에서 Lawrence D. Miles에 의해 체계화되었으며, 기능 중심 사고(Function-Oriented Thinking)를 핵심 철학으로 한다.

[^2]: FAST(Function Analysis System Technique)는 기능 간의 "왜(Why)-어떻게(How)" 관계를 분석하여 기능구조를 시각화하는 대표적인 기능분석 기법이다.

[^3]: 목표원가관리(Target Costing)는 시장가격에서 목표이익을 차감하여 목표원가를 설정하고, 이를 달성하기 위한 핵심 수단으로 VE를 활용한다.

[^4]: 국내 공공건설사업에서는 일정 규모 이상의 사업에 대해 설계 VE를 의무적으로 수행하는 경우가 많으며, 원가절감뿐 아니라 품질 및 안전성 향상을 동시에 추구한다.

[^5]: 최근 가치공학은 비용 절감 중심에서 고객경험, ESG, 탄소중립, 순환경제까지 포함하는 지속가능 가치혁신(Sustainable Value Innovation) 개념으로 확대되고 있다.
