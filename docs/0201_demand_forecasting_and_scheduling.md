## 수요예측과 일정관리

수요예측(Forecasting)은 미래 고객수요를 예측하여 생산계획, 재고계획, 인력계획 및 공급망 운영의 기준을 제공하는 활동이다. 일정관리(Scheduling)는 수요예측 결과를 기반으로 생산활동을 구체적인 작업 일정으로 전환하여 납기, 원가, 생산성을 관리하는 활동이다. 최근에는 AI 기반 수요예측과 APS(Advanced Planning & Scheduling)를 활용한 실시간 생산계획이 확대되고 있다.

### 수요예측의 개요

#### 수요예측의 정의

수요예측이란 과거 데이터와 시장정보를 활용하여 미래 수요를 추정하는 과정이다.

#### 수요예측의 목적

* 생산계획 수립
* 재고 최적화
* 설비능력 계획
* 인력계획 수립
* 공급망 안정화

#### 수요예측의 중요성

수요예측은 생산운영의 출발점이며 예측 정확도는 재고, 납기, 생산성 및 수익성에 직접적인 영향을 미친다.

```text
수요예측
↓
생산계획
↓
자재계획
↓
생산일정
↓
납기관리
```

### 수요의 특성

수요는 다양한 패턴을 가지며 적절한 예측기법 선택의 기준이 된다.

#### 수요의 구성요소

| 구성요소              | 의미               |
| ----------------- | ---------------- |
| 수평변동(Horizontal)  | 일정 수준 중심의 변동     |
| 추세변동(Trend)       | 장기적 증가 또는 감소     |
| 계절변동(Seasonality) | 일정 주기로 반복되는 변동   |
| 순환변동(Cycle)       | 경기변동 등에 의한 장기 변동 |
| 불규칙변동(Random)     | 예측 불가능한 변동       |

#### 수요패턴과 예측기법

| 수요패턴 | 적합 기법         |
| ---- | ------------- |
| 수평변동 | 이동평균, 지수평활    |
| 추세변동 | Holt 모형, 회귀분석 |
| 계절변동 | Holt-Winters  |
| 복합패턴 | ARIMA         |

### 수요예측 프로세스

#### 수요예측 절차

```text
데이터 수집
↓
수요특성 분석
↓
예측기법 선정
↓
수요예측 수행
↓
오차 측정
↓
생산계획 반영
↓
지속적 개선
```

#### 예측 시 유의사항

* 예측은 항상 오차를 수반한다.
* 예측기간이 길수록 정확도가 낮아진다.
* 여러 기법을 병행하여 검증한다.
* 정량적 자료와 정성적 정보를 함께 활용한다.

### 수요예측 기법

수요예측 기법은 정성적 기법과 정량적 기법으로 구분된다.

### 정성적 예측기법

정성적 예측은 과거 데이터가 부족하거나 신제품 수요예측에 활용된다.

#### 주요 기법

| 기법      | 특징          | 적용 분야  |
| ------- | ----------- | ------ |
| 델파이법    | 전문가 반복 의견수렴 | 장기예측   |
| 중역판단법   | 경영진 경험 활용   | 신제품    |
| 판매원 의견법 | 현장정보 활용     | 지역별 수요 |
| 소비자 조사법 | 고객 구매의향 조사  | 신제품 개발 |

#### 델파이법

전문가 집단의 의견을 익명으로 반복 수집하여 합의를 도출하는 방법이다.

##### 장점

* 장기예측 가능
* 전문가 지식 활용
* 집단사고 방지

##### 단점

* 시간 소요
* 조사비용 증가

### 정량적 예측기법

정량적 예측은 과거 데이터를 통계적으로 분석하여 미래 수요를 예측하는 방법이다.

#### 시계열 분석법

##### 이동평균법(Moving Average)

최근 일정 기간의 평균을 이용하는 방법

###### 특징

* 계산이 간단
* 단기예측 적합
* 추세변화 반영 한계

##### 가중이동평균법

최근 데이터에 높은 가중치를 부여하는 방법

##### 지수평활법(Exponential Smoothing)

최근 데이터에 지수적으로 높은 가중치를 부여하는 방법

###### 특징

* 계산이 용이
* 단기예측 적합
* 실무 활용도 높음

##### Holt 모형

추세를 반영한 지수평활모형

##### Holt-Winters 모형

추세와 계절성을 동시에 반영하는 모형

##### ARIMA 모형

자기회귀와 이동평균을 결합한 시계열 예측모형

#### 인과모형(Causal Model)

##### 회귀분석

수요와 영향요인 간의 관계를 분석하여 예측

예시

* 광고비와 판매량
* GDP와 자동차 수요
* 금리와 주택 수요

##### 선도지표법

미래 수요를 선행적으로 설명하는 지표를 활용하는 방법

### 예측기법 선정기준

| 상황        | 적합 기법        |
| --------- | ------------ |
| 과거 데이터 부족 | 델파이법         |
| 신제품 출시    | 소비자조사        |
| 안정적 수요    | 이동평균         |
| 추세 존재     | Holt 모형      |
| 계절성 존재    | Holt-Winters |
| 원인 분석 필요  | 회귀분석         |

### 예측오차 관리

수요예측은 불확실성을 내포하므로 오차관리가 중요하다.

#### 예측오차

```text
예측오차
=
실제수요 - 예측수요
```

### 오차 측정지표

#### MAD

평균 절대오차

```text
MAD =
Σ|오차|
────────
n
```

#### MSE

평균제곱오차

```text
MSE =
Σ(오차²)
────────
n
```

#### RMSE

평균제곱근오차

```text
RMSE = √MSE
```

#### MAPE

평균 절대 백분율 오차

```text
MAPE =
Σ(|오차|/실제값)
───────────── ×100
n
```

#### Tracking Signal(TS)

예측의 편향 여부를 판단하는 지표

```text
TS =
RSFE
────
MAD
```

일반적으로 ±4 범위를 벗어나면 예측모형 재검토가 필요하다.

#### 오차지표 비교

| 지표   | 특징       |
| ---- | -------- |
| MAD  | 이해가 쉬움   |
| MSE  | 큰 오차 강조  |
| RMSE | 실제 단위 사용 |
| MAPE | 비율 비교 가능 |
| TS   | 편향 검출    |

### 수요불확실성 대응

#### 채찍효과(Bullwhip Effect)

공급망 상류로 갈수록 수요변동이 확대되는 현상이다.

```text
소비자
↓
유통업체
↓
도매업체
↓
제조업체
↓
공급업체

수요변동 확대
```

##### 원인

* 정보 왜곡
* 과도한 안전재고
* 일괄 주문
* 가격 프로모션

##### 대응방안

* 정보공유
* 공급망 협업
* 실시간 수요정보 활용

#### Postponement

제품 차별화를 가능한 한 늦추는 전략

#### QR(Quick Response)

실수요 정보를 활용하여 신속 대응하는 전략

#### ECR(Efficient Consumer Response)

유통망 전체의 효율성을 향상시키는 전략

#### VMI(Vendor Managed Inventory)

공급자가 고객 재고를 직접 관리하는 방식

!!! note "관련 단원"

```
VMI와 공급망 협업은 「공급망 관리」에서 상세히 설명한다.
```

## 일정관리(Scheduling)

일정관리는 생산계획을 작업 수준으로 구체화하여 납기와 생산성을 동시에 확보하는 활동이다.

### 일정관리의 목적

* 납기 준수
* 재고 감소
* 설비 활용도 향상
* 생산성 향상
* 리드타임 단축

### 생산계획과 일정관리의 관계

```text
수요예측
↓
생산계획(MPS)
↓
자재소요계획(MRP)
↓
능력소요계획(CRP)
↓
일정계획(Scheduling)
↓
현장통제(SFC)
```

### 일정계획 방식

#### Forward Scheduling

현재 시점부터 가능한 한 빨리 작업을 시작하는 방식

##### 장점

* 설비 활용도 향상
* 일정 수립 용이

##### 단점

* 재공품 증가
* 재고 증가 가능

#### Backward Scheduling

납기를 기준으로 역산하여 계획하는 방식

##### 장점

* 재고 최소화
* 납기 중심 운영

##### 단점

* 일정변동 대응 어려움

#### Forward Scheduling과 Backward Scheduling 비교

| 구분 | Forward | Backward |
| -- | ------- | -------- |
| 기준 | 현재 시점   | 납기 시점    |
| 목적 | 설비 활용   | 납기 준수    |
| 재고 | 증가 가능   | 최소화      |
| 활용 | MTS     | MTO      |

### 능력소요계획(CRP)

CRP(Capacity Requirements Planning)는 생산계획을 수행하기 위한 설비 및 인력 능력을 검토하는 활동이다.

#### 목적

* 병목공정 확인
* 설비부하 조정
* 생산능력 확보

## 최근 동향

### S&OP

S&OP(Sales & Operations Planning)는 판매계획과 생산계획을 통합하는 경영 프로세스이다.

```text
수요계획
↓
판매계획
↓
생산계획
↓
공급계획
↓
재무계획
```

### APS

APS(Advanced Planning & Scheduling)는 제약조건을 고려하여 생산계획과 일정을 최적화하는 시스템이다.

#### 특징

* 실시간 계획 수립
* 병목공정 반영
* 최적 생산일정 생성

### Demand Driven MRP

실제 수요를 기반으로 자재계획을 수행하는 방식

### AI 기반 수요예측

머신러닝과 딥러닝을 활용하여 대규모 데이터를 분석하고 수요를 예측하는 방식이다.

#### 활용 기술

* 시계열 머신러닝
* 딥러닝
* 생성형 AI
* 디지털 트윈

## 수요예측과 일정관리의 종합

수요예측은 생산활동의 출발점이며 일정관리는 이를 실행 가능한 생산계획으로 구체화하는 과정이다. 기업은 예측 정확도를 향상시키고 생산능력과 일정을 최적화함으로써 재고를 최소화하면서도 고객 납기를 만족시켜야 한다. 최근에는 AI, APS, S&OP를 활용한 데이터 기반 생산운영체계가 경쟁력의 핵심요소로 자리잡고 있다.
