## 사업환경 분석 개요
사업환경 분석은 외부 환경과 내부 환경으로 구성한다.

사업환경 분석 개요
: - 외부 환경(거시환경, 산업환경)과 내부 환경 구분
- 사업환경 분석의 중요성
- 분석 도구 활용 시 유의사항

## 사업환경 분석 도구
### 5-Forces Model
5-Forces Model은 산업 내 경쟁환경 분석 도구다.

5-Forces Model
: - 산업 내 경쟁강도, 신규진입자 위협, 대체재 위협, 공급자 교섭력, 구매자 교섭력
- 장점과 한계
- 삼성전자 스마트폰 적용 사례

### SWOT 분석
SWOT 분석은 내부와 외부 환경을 종합하는 도구다.

SWOT 분석
: - 강점, 약점, 기회, 위협 구성
- 장점과 단점

**(추가 권장)** SO/ST/WO/WT 전략 매트릭스

### BCG 매트릭스
BCG 매트릭스는 사업 포트폴리오 분석 도구다.

BCG 매트릭스
: - 스타, 캐시카우, 물음표, 도그 구성
- 활용 방안과 한계

### GE 매트릭스
GE 매트릭스는 BCG의 확장형 포트폴리오 분석 도구다.

GE 매트릭스
: - 시장성장률과 경쟁력 기준 9개 영역 구성
- 영역별 전략

**(추가 권장)** BCG 매트릭스와 GE 매트릭스 비교표

### PEST 분석
PEST 분석은 거시환경 분석 도구다.

PEST 분석
: - 정치, 경제, 사회, 기술 요인
- 주요 목적

### 가치사슬 다이어그램
가치사슬 다이어그램은 내부 활동 분석 도구다.

가치사슬 다이어그램
: - 기본 활동과 지원 활동 구성
- 분석 절차와 활용

**(상호참조 권장)** 0101_competitive_manufacturing_strategy.md 본원적 경쟁전략과의 연계

### BPR
BPR은 프로세스 재설계를 통한 성과개선 방법론이다.

BPR
: - 근본적 재설계, 프로세스 중심, IT 활용, 고객 중심 개념
- 주요 단계와 장단점
- 업종별 적용 사례

**(주의)** 상세 실행기법은 0306_process_improvement.md, 0704_production_innovation_technology.md에서 심화 서술 권장

## 입지 분석
### 국내 공장입지
국내 공장입지는 선정 요소와 절차로 구성한다.

국내 공장입지
: - 시장 접근성, 원재료 공급, 교통 인프라, 노동력, 토지비용 등 선정 요소
- 요구조건 분석부터 최종선정까지의 절차

### 해외 공장입지
해외 공장입지는 국내 입지와 정치·경제적 고려요소가 추가된다.

해외 공장입지
: - 시장접근성, 생산비용, 정치·경제 안정성 등 선정 요소
- 시장분석부터 투자실행까지의 절차

### 입지 평가방법
입지 평가방법은 정성적 기법, 정량적 기법, 종합 의사결정 기법으로 구분한다.

입지 평가방법
: - 체크리스트법과 평가행렬법은 정성적 평가기법
- 중심지법, 수송모형, 손익분기점 분석은 정량적 평가기법
- AHP, MCDM은 종합 의사결정 기법
