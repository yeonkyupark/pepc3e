## 생산 계획 구조
생산계획은 장기, 중기, 단기 계획으로 있다.

생산 계획 구조
: - 장기계획은 기업전략과 재무계획 수립
- 중기계획은 총괄계획과 자재계획 수립
- 단기계획은 분기, 월별 생산계획 수립

## 일정계획
일정계획은 대일정, 중일정, 소일정 계획으로 있다.

일정계획
: - 대일정계획은 수주부터 출하까지의 계획
- 중일정계획은 공정 개시일과 완료일 계획
- 소일정계획은 작업자와 기계별 세부 계획

**(보강 권장)** 우선순위 규칙(SPT, EDD 등) 계산 예제

### 공정관리 기능
공정관리는 계획기능과 통제기능으로 구성

공정관리 기능
: - 계획기능은 공정계획, 공수계획, 일정계획
- 통제기능은 작업지시, 능력관리, 진도관리

## 총괄생산계획(APP)
총괄생산계획은 중기 수요에 대응하는 산출물 공급계획

총괄생산계획
: - 목표는 이익극대화와 비용최소화
- 총괄과 분해를 통한 계획 수립
- 수요추종, 생산평준화, 혼합 전략

### RRP
RRP는 중장기 자원 용량을 총괄적으로 계획하는 시스템

RRP
: - S&OP와 MPS 사이에 위치
- 자원부족 예측과 투자계획 지원

## 기준일정계획(MPS)
기준일정계획은 개별 최종품목의 단기생산계획

기준일정계획
: - Frozen, Slushy, Open 구간 구분
- ATP를 통한 약속가능수량 산출

### RCCP
RCCP는 MPS 기준 자원 용량을 대략적으로 검토

RCCP
: - RRP와 CRP 사이에 위치
- 병목자원 조기식별과 계획조정

## 자재소요계획(MRP)
자재소요계획은 원자재 및 부품 수요 계획 시스템

자재소요계획
: - MPS, BOM, IRF를 입력요소로 사용
- 자재소요량계획, 주문계획 등을 산출

### CRP
CRP는 작업지시 기반 세부 자원소요를 정밀 산출

CRP
: - 작업장별 자원부하 산출과 관리
- RCCP 대비 상세하고 단기적인 계획
