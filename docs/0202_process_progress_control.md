현재 문서는 전반적으로 **공정관리 기법 중심**으로 정리되어 있지만, 기술사 관점에서는 **"생산계획 → 작업지시 → 진도관리 → 통제 → 개선"**이라는 운영 흐름이 조금 약한 편입니다.

또한 프로세스 진도관리는 단순히 진척도를 확인하는 것이 아니라,

* 납기관리
* 재공품 관리
* 병목관리
* 생산통제
* 현장 모니터링

까지 포함하는 개념으로 이해하는 것이 좋습니다.

추천 구조는 다음과 같습니다.

```text
생산계획
↓
작업지시
↓
진도관리
↓
공정통제
↓
병목관리
↓
생산성과 납기관리
↓
스마트 생산통제
```

# 프로세스 진도관리

프로세스 진도관리(Progress Control)는 생산계획에 따라 작업이 예정된 일정과 수량대로 수행되고 있는지를 관리하는 활동이다. 진도관리는 생산통제의 핵심 기능으로서 납기 준수, 재공품 감소, 생산성 향상 및 고객만족 확보를 목적으로 한다.

## 프로세스 진도관리의 개요

### 진도관리의 정의

진도관리란 생산계획과 실제 작업실적을 비교하여 차이를 분석하고 필요한 조치를 수행하는 관리활동이다.

### 진도관리의 목적

* 납기 준수
* 생산계획 달성
* 재공품 감소
* 생산성 향상
* 병목공정 관리
* 고객서비스 향상

### 진도관리의 중요성

생산계획이 아무리 우수하더라도 실행단계에서 진도관리가 이루어지지 않으면 납기지연, 재고증가 및 생산성 저하가 발생한다.

```text
생산계획
↓
작업지시
↓
진도관리
↓
실적수집
↓
차이분석
↓
시정조치
```

## 생산통제와 진도관리

### 생산통제의 개념

생산통제(Production Control)는 생산활동이 계획대로 수행되도록 관리하는 기능이다.

### 생산통제의 기능

| 기능   | 내용         |
| ---- | ---------- |
| 작업지시 | 생산작업 배정    |
| 진도관리 | 진행상황 파악    |
| 일정조정 | 일정 변경 및 대응 |
| 실적관리 | 생산실적 분석    |
| 시정조치 | 문제 해결      |

### 생산통제 체계

```text
계획(Plan)
↓
실행(Do)
↓
점검(Check)
↓
개선(Action)
```

진도관리는 PDCA 사이클의 Check 단계에 해당한다.

## 진도관리 방식

### 집중식 진도관리

중앙관리부서에서 전체 생산진도를 통합 관리하는 방식

#### 장점

* 전체 최적화 가능
* 통제력 우수
* 정보 일관성 확보

#### 단점

* 현장 대응 지연
* 관리부하 증가

### 분산식 진도관리

각 공정 또는 작업조직이 직접 진도를 관리하는 방식

#### 장점

* 신속한 대응
* 현장 자율성 향상

#### 단점

* 부서 간 조정 어려움
* 전체 최적화 한계

### 집중식과 분산식 비교

| 구분   | 집중식 | 분산식 |
| ---- | --- | --- |
| 통제력  | 높음  | 낮음  |
| 대응속도 | 느림  | 빠름  |
| 정보통합 | 우수  | 제한적 |
| 자율성  | 낮음  | 높음  |

## 작업지시와 진도관리

### 작업지시(Dispatching)

작업지시는 생산계획을 실제 작업으로 전환하는 활동이다.

### 작업지시서 구성항목

* 작업번호
* 품목명
* 생산수량
* 작업순서
* 작업장
* 납기일

### 작업배정 원칙

* 설비능력 고려
* 작업부하 균형화
* 병목 최소화
* 납기 우선 고려

## 진도관리 기법

### 간트차트(Gantt Chart)

#### 개요

작업일정과 진행상황을 막대그래프로 표현하는 기법이다.[^1]

#### 장점

* 이해가 용이
* 일정관리 편리
* 진척도 확인 가능

#### 단점

* 복잡한 프로젝트 적용 한계
* 작업 간 상호관계 표현 부족

```text
작업A ██████████

작업B     ███████

작업C         ███████
```

### PERT

#### 개요

불확실성이 높은 프로젝트 일정계획 기법이다.[^2]

#### 특징

* 확률적 접근
* 프로젝트 관리 적합
* 일정 위험도 분석 가능

#### 3점 추정

```text
기대시간 =
(a + 4m + b)
──────────
     6
```

| 기호 | 의미   |
| -- | ---- |
| a  | 낙관시간 |
| m  | 최빈시간 |
| b  | 비관시간 |

### CPM

#### 개요

프로젝트 완료시간을 결정하는 주공정을 찾는 기법이다.[^3]

#### 목적

* 최단 공기 산정
* 중요 작업 관리
* 일정 단축

#### 주공정(Critical Path)

여유시간(Slack)이 0인 경로

```text
Start
 ↓
 A
 ↓
 B
 ↓
 C
 ↓
Finish
```

#### PERT와 CPM 비교

| 구분 | PERT | CPM   |
| -- | ---- | ----- |
| 시간 | 확률적  | 결정적   |
| 적용 | 연구개발 | 건설·생산 |
| 목적 | 일정예측 | 일정통제  |

[^1]: Henry L. Gantt가 개발한 일정관리 기법.

[^2]: Program Evaluation and Review Technique.

[^3]: Critical Path Method.

## 공정 진척도 측정

### 진척도 관리

생산계획 대비 실제 수행 수준을 측정하는 활동

#### 주요 지표

| 지표    | 의미         |
| ----- | ---------- |
| 계획진도  | 계획된 작업량    |
| 실제진도  | 실제 수행량     |
| 진도율   | 실제/계획      |
| 납기준수율 | 기한 내 완료 비율 |

### 진도차이 분석

```text
진도차이
=
실제진도
-
계획진도
```

### 시정조치

* 작업 우선순위 조정
* 추가 인력 투입
* 설비 증설
* 외주 활용

## 병목관리(TOC)

### 개요

TOC(Theory of Constraints)는 시스템 성과를 제한하는 제약요인을 관리하는 기법이다.[^4]

### 기본 개념

```text
병목공정
↓
전체 생산능력 결정
```

### TOC 5단계

1. 제약요인 발견
2. 제약요인 활용
3. 비제약 공정 동기화
4. 제약요인 개선
5. 새로운 제약 탐색

### Drum-Buffer-Rope

| 요소     | 의미         |
| ------ | ---------- |
| Drum   | 병목공정 기준 생산 |
| Buffer | 완충재고       |
| Rope   | 생산동기화      |

!!! note "관련 단원"

```
TOC는 「공정개선」에서도 심화 학습한다.
```

[^4]: Eliyahu Goldratt, *The Goal*, 1984.

## 납기관리

### 납기관리의 정의

고객 요구 납기를 만족시키기 위해 생산일정을 관리하는 활동

### 납기관리 지표

| 지표     | 의미       |
| ------ | -------- |
| 납기준수율  | 정시 납품 비율 |
| 평균지연일수 | 납기 초과 일수 |
| 리드타임   | 주문~납품 기간 |

### 납기지연 원인

* 병목공정 발생
* 설비고장
* 자재부족
* 수요급증

### 대응방안

* APS 활용
* 생산능력 조정
* 안전재고 운영
* 공급망 협업

## 스마트 진도관리

### MES 기반 진도관리

MES는 생산현장의 작업실적을 실시간 수집·관리하는 시스템이다.

#### 주요 기능

* 작업지시
* 생산실적 수집
* 설비상태 모니터링
* 품질정보 관리

### APS 기반 일정관리

APS는 설비능력과 제약조건을 고려하여 최적 일정을 수립한다.

### 디지털 전환과 진도관리

최근에는 IoT, AI, 디지털 트윈을 활용한 실시간 생산통제가 확대되고 있다.

| 기술     | 활용 내용         |
| ------ | ------------- |
| IoT    | 실시간 데이터 수집    |
| AI     | 생산예측 및 일정 최적화 |
| 디지털 트윈 | 가상 생산시뮬레이션    |
| CPS    | 자율 생산제어       |

## 프로세스 진도관리의 종합

프로세스 진도관리는 생산계획을 현장에서 실행하고 통제하는 핵심 관리기능이다. 작업지시, 진척도 측정, 병목관리 및 납기관리를 통해 생산성과 고객만족을 확보하며, 최근에는 MES·APS·AI 기반 스마트 생산통제로 발전하고 있다. 특히 진도관리는 생산계획과 현장운영을 연결하는 핵심 고리로서 공장관리의 실질적 실행력을 결정하는 요소이다.
