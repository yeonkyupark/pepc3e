## 진도관리
공정진도관리는 목적, 방법, 도구로 구성

공정진도관리 개요
: - 납기준수, 생산흐름 최적화, 재고비용 절감, 자원활용 극대화 목적
- 일정계획법, MRP, TOC 기반 관리, 진도통제, JIT 방식
- 간트차트, PERT/CPM, ERP 시스템 등 주요 도구

## 네트워크 기법
### 주경로
주경로는 여유시간이 최소가 되는 단계를 연결한 경로

주경로
: - 여유시간 최소 경로가 주경로(애로공정)
- 전진계산과 후진계산을 통한 여유시간 산출

### 네트워크 구성
네트워크 구성은 AOA 방식과 AON 방식으로 구분

네트워크 구성
: - 단계, 활동, 결합점, 호 등 구성요소
- AOA는 활동을 화살표로 표시
- AON은 활동을 마디로 표시

**(이동 권장)** 라인편성효율(사이클타임, 최소공정수) 예제는 0102_production_structure_and_layout.md 택트타임 항목과 통합 또는 상호참조

## 주요 기법
### PERT
PERT는 확률적 모델 기반의 일정관리 기법

PERT
: - 낙관치, 정상치, 비관치를 활용한 기대시간 산출
- 장점과 단점

### CPM
CPM은 확정적 모델 기반의 일정관리 기법

CPM
: - 정상시간과 단축시간을 활용한 비용계산
- 장점과 단점

### CCM
CCM은 TOC 제약이론 기반의 프로젝트 관리 기법

CCM
: - 안전여유, 자기방어, 학생증후군, 파킨슨법칙 등 지연원인
- 프로젝트 버퍼, 피딩 버퍼, 자원 버퍼
- 수행절차와 일정수립절차

### 일정 단축 기법
일정 단축 기법은 크래싱과 Fast Tracking으로 구성

일정 단축 기법
: - 크래싱은 자원 추가 투입을 통한 일정단축
- Fast Tracking은 병렬작업을 통한 일정단축

## 기법 간 비교
PERT, CPM, CCM은 개발배경과 관리방식에 차이

기법 간 비교
: - PERT vs CPM 비교(모델, 관리대상, 주요대상)
- CPM vs CCM 비교(착수일, 관리관점, 버퍼, 자원제약)

**(추가 권장)** PERT·CPM·CCM 통합 비교표

## 참고자료
### 프로젝트와 프로세스 차이
프로젝트와 프로세스는 시작종료, 목표, 범위 등에서 차이

프로젝트와 프로세스 차이
: - 프로젝트는 명확한 시작과 끝, 한정된 범위
- 프로세스는 계속적 진행, 상대적으로 넓은 범위

### PERT/COST
PERT/COST는 최소비용 완료를 위한 일정계획 기법

PERT/COST
: - 직접비용과 간접비용 구성
- 작업분해도부터 예산화까지의 분석절차
