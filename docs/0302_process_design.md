## 공정설계

공정설계(Process Design)는 고객 요구사항을 만족하는 제품과 서비스를 효율적으로 생산하기 위하여 생산공정의 구조, 작업순서, 설비 및 자원 배치를 체계적으로 설계하는 활동이다. 공정설계는 생산성, 품질, 원가, 납기 및 유연성을 결정하는 핵심 요소로서 생산시스템 구축의 출발점이 된다.

### 공정설계의 개요

#### 공정설계의 정의

공정설계란 제품이나 서비스를 생산하기 위한 작업순서, 공정구조, 설비 및 자원 활용방식을 결정하는 활동이다.

#### 공정설계의 목적

* 생산성 향상
* 품질 확보
* 원가 절감
* 납기 단축
* 유연성 확보
* 안전성 향상

#### 공정설계의 중요성

공정설계 단계에서 결정된 사항은 향후 생산비용, 품질수준 및 운영효율에 큰 영향을 미친다.

```text
제품설계
    ↓
공정설계
    ↓
설비계획
    ↓
생산계획
    ↓
생산운영
```

## 공정설계 절차

### 공정설계 프로세스

공정설계는 제품정보를 생산시스템으로 구현하는 과정이다.

```text
제품분석
    ↓
공정선정
    ↓
작업순서 결정
    ↓
설비선정
    ↓
공정배치
    ↓
생산성 검증
```

### 주요 검토사항

#### 제품 특성

* 제품구조
* 품질요구사항
* 생산수량
* 제품 다양성

#### 생산자원

* 설비능력
* 인력수준
* 기술수준
* 투자규모

#### 운영요건

* 납기
* 원가
* 품질
* 유연성

## 공정선택(Process Selection)

### 공정선택의 개요

공정선택은 제품 특성과 생산량에 적합한 생산방식을 결정하는 활동이다.

### 제품-공정 매트릭스

제품 특성과 공정 유형의 적합성을 설명하는 대표적인 모델이다.[^1]

| 생산방식             | 생산량   | 제품 다양성 | 공정특성   |
| ---------------- | ----- | ------ | ------ |
| 프로젝트             | 매우 낮음 | 매우 높음  | 위치고정형  |
| 잡샵(Job Shop)     | 낮음    | 높음     | 공정별 배치 |
| 배치생산(Batch)      | 중간    | 중간     | 기능별 생산 |
| 조립라인(Line)       | 높음    | 낮음     | 제품별 배치 |
| 연속생산(Continuous) | 매우 높음 | 매우 낮음  | 자동화 공정 |

### 생산량과 다양성의 관계

```text
제품 다양성 ↑

프로젝트
   ↓
잡샵
   ↓
배치생산
   ↓
라인생산
   ↓
연속생산

생산량 ↑
```

[^1]: Hayes & Wheelwright의 Product-Process Matrix.

## 공정분석

### 공정분석의 개요

공정분석은 생산활동을 체계적으로 분석하여 낭비를 제거하고 생산성을 향상시키기 위한 기법이다.

### 공정분석의 목적

* 작업 개선
* 공정 단순화
* 낭비 제거
* 생산성 향상

### 공정분석의 기본 질문

공정개선은 다음 질문에서 시작된다.

* 왜 수행하는가?
* 무엇을 수행하는가?
* 어디서 수행하는가?
* 언제 수행하는가?
* 누가 수행하는가?
* 어떻게 수행하는가?

### ECRS 원칙

공정개선의 기본 원칙이다.

| 원칙        | 내용  |
| --------- | --- |
| Eliminate | 제거  |
| Combine   | 결합  |
| Rearrange | 재배치 |
| Simplify  | 단순화 |

!!! note "관련 단원"

```

```

## 공정도(Process Chart)

### 공정도의 개요

공정도는 작업 및 물자의 흐름을 기호로 표현한 분석도구이다.

### ASME 공정분석 기호

| 기호 | 의미                 |
| -- | ------------------ |
| ○  | 작업(Operation)      |
| □  | 검사(Inspection)     |
| →  | 운반(Transportation) |
| D  | 대기(Delay)          |
| ▽  | 저장(Storage)        |

### 공정도 활용 목적

* 작업흐름 파악
* 병목공정 분석
* 낭비 제거
* 표준화 지원

## 흐름분석

### 흐름분석의 개요

공정 내 자재와 작업자의 이동경로를 분석하는 기법이다.

### 주요 분석도구

#### Flow Process Chart

작업 흐름을 상세히 기록하는 도표

#### Flow Diagram

실제 작업장의 이동경로를 도면에 표시

#### String Diagram

이동거리와 동선을 실선으로 분석

#### Travel Chart

부서 간 이동빈도 분석

### 흐름분석의 목적

* 운반거리 단축
* 작업동선 개선
* 물류비 절감
* 설비배치 개선

## 병목공정 관리

### 병목공정의 정의

병목(Bottleneck)이란 생산능력이 부족하여 전체 공정의 처리능력을 제한하는 공정을 의미한다.

### 병목공정의 영향

* 생산량 감소
* 리드타임 증가
* 재공품 증가
* 납기 지연

### 병목공정 개선방안

* 설비 증설
* 작업방법 개선
* 외주 활용
* 우선순위 조정

### TOC 관점의 병목관리

골드렛(Goldratt)의 제약이론(TOC)은 병목공정을 중심으로 생산시스템을 개선한다.[^2]

```text
제약 발견
    ↓
제약 활용
    ↓
제약 집중
    ↓
능력 향상
    ↓
반복 개선
```

[^2]: Theory of Constraints.

!!! note "관련 단원"

```
TOC와 DBR 기법은 「공정개선」에서 상세히 설명한다.
```

## 동시공학(Concurrent Engineering)

### 개요

제품개발과 공정설계를 동시에 수행하여 개발기간을 단축하는 접근방법이다.

### 특징

* 부서 간 협업
* 설계와 생산 통합
* 개발기간 단축
* 품질 향상

### 기대효과

* 시장출시기간(Time to Market) 단축
* 설계변경 감소
* 생산성 향상

## DFM과 DFA

### DFM

DFM(Design for Manufacturing)은 생산 용이성을 고려한 제품설계 기법이다.

#### 목적

* 제조원가 절감
* 생산성 향상
* 품질 안정화

### DFA

DFA(Design for Assembly)는 조립 용이성을 고려한 설계기법이다.

#### 목적

* 부품수 감소
* 조립시간 단축
* 조립품질 향상

### DFM과 DFA 비교

| 구분 | DFM    | DFA     |
| -- | ------ | ------- |
| 목적 | 제조 용이성 | 조립 용이성  |
| 관점 | 가공 중심  | 조립 중심   |
| 효과 | 원가 절감  | 리드타임 단축 |

## FMEA

### 개요

FMEA(Failure Mode and Effects Analysis)는 잠재적 고장유형을 사전에 분석하여 예방하는 기법이다.

### 분석 절차

```text
기능 정의
    ↓
고장유형 도출
    ↓
영향 분석
    ↓
원인 분석
    ↓
위험 우선순위 결정
    ↓
개선대책 수립
```

### 평가요소

| 항목         | 의미  |
| ---------- | --- |
| Severity   | 심각도 |
| Occurrence | 발생도 |
| Detection  | 검출도 |

### 활용 효과

* 품질 향상
* 신뢰성 확보
* 설계 리스크 감소

!!! note "관련 단원"

```
품질개선 및 예방품질관리 기법은 「품질관리」에서 상세히 설명한다.
```

## 공정설계의 종합

공정설계는 제품을 효율적으로 생산하기 위한 생산시스템의 구조를 결정하는 핵심 활동이다. 제품 특성에 적합한 공정을 선택하고, 공정분석과 흐름분석을 통해 낭비를 제거하며, 병목관리와 예방적 품질설계를 통해 생산성과 품질을 동시에 확보한다. 최근에는 스마트팩토리와 디지털 전환의 확산에 따라 자동화, AI 및 디지털 트윈을 고려한 공정설계가 경쟁력 확보의 핵심 요소로 자리잡고 있다.

### 범위 연계

| 공정설계 요소        | 관련 단원    |
| -------------- | -------- |
| 설비배치(Layout)   | 생산구조와 방식 |
| 생산계획(MRP, CRP) | 생산계획수립   |
| 표준작업           | 표준작업관리   |
| TOC            | 공정개선     |
| FMEA           | 품질관리     |
| 스마트 제조         | 정보화활용기술  |
