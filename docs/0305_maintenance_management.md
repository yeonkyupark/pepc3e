# 설비보전관리

설비보전관리(Equipment Maintenance Management)는 생산설비의 기능을 정상 상태로 유지하거나 신속히 복원하여 생산성을 극대화하는 관리활동이다. 과거의 사후보전(BM) 중심에서 예방보전(PM), 생산보전(Productive Maintenance), TPM(Total Productive Maintenance), 신뢰성 중심 보전(RCM), 상태기반 보전(CBM), 예지보전(PdM)으로 발전하고 있으며, 최근에는 AI와 IoT를 활용한 스마트보전으로 진화하고 있다. ([낟알 데이터][1])

## 설비보전관리의 개요

### 설비보전의 정의

설비보전(Maintenance)이란 설비의 기능을 유지하거나 고장 발생 시 신속히 복구하여 생산활동을 안정적으로 수행하도록 하는 일련의 관리활동이다.

### 설비보전의 목적

* 설비 신뢰성 향상
* 생산성 향상
* 품질 안정화
* 설비수명 연장
* 보전비 절감
* 안전사고 예방

### 설비보전의 기본 개념

```text
설비 열화
     ↓
이상 징후 발견
     ↓
적절한 보전 수행
     ↓
설비 성능 유지
     ↓
생산성 및 품질 확보
```

## 설비의 열화와 손실

### 설비 열화

설비는 시간 경과와 사용에 따라 성능이 저하되며 이를 열화(Deterioration)라 한다.

### 열화의 종류

| 구분   | 특징               | 대응방법       |
| ---- | ---------------- | ---------- |
| 자연열화 | 시간 경과에 따른 성능 저하  | 예방보전       |
| 강제열화 | 사용·관리 부주의에 따른 열화 | 자주보전, 표준작업 |

### 설비 로스

#### 돌발로스

예측하기 어려운 갑작스러운 고장

* 설비 고장
* 부품 파손
* 긴급 정지

#### 만성로스

반복적으로 발생하는 작은 손실

* 미세정지
* 속도 저하
* 품질 편차

> **돌발로스는 신속한 복구가 중요하며, 만성로스는 원인 제거를 통한 지속적 개선이 핵심이다.** ([낟알 데이터][1])

## 설비보전 방식

### 보전 방식의 발전

```text
BM
(사후보전)
      ↓
PM
(예방보전)
      ↓
TPM
(생산보전)
      ↓
RCM
(신뢰성 중심)
      ↓
CBM
(상태기반)
      ↓
PdM
(예지보전)
```

### 사후보전(BM)

Breakdown Maintenance

고장 발생 후 수리하는 방식

#### 장점

* 관리가 단순
* 초기 비용이 적음

#### 단점

* 생산중단 발생
* 품질 저하
* 긴급수리 비용 증가

### 예방보전(PM)

Preventive Maintenance

일정 주기에 따라 계획적으로 수행하는 보전

#### 장점

* 고장 예방
* 설비 신뢰성 향상

#### 단점

* 과잉보전 가능
* 보전비 증가

### 개량보전(CM)

Corrective Maintenance

설비 구조를 개선하여 고장을 근본적으로 감소시키는 활동

### 보전예방(MP)

Maintenance Prevention

설비 설계 단계에서 보전성과 신뢰성을 확보하는 활동

### 상태기반보전(CBM)

Condition Based Maintenance

설비 상태를 실시간으로 진단하여 필요한 경우에만 보전을 수행하는 방식

### 예지보전(PdM)

Predictive Maintenance

AI와 IoT를 활용하여 고장을 사전에 예측하고 최적 시점에 보전하는 방식

### 보전방식 비교

| 구분  | 보전 시점   | 특징     |
| --- | ------- | ------ |
| BM  | 고장 후    | 반응형    |
| PM  | 정기 주기   | 예방형    |
| CBM | 상태 변화 시 | 상태 기반  |
| PdM | 고장 예측 시 | 데이터 기반 |
| CM  | 설비 개선 시 | 근본 개선  |

## TPM(Total Productive Maintenance)

### TPM의 개요

TPM은 생산부문과 보전부문을 포함한 전 구성원이 참여하여 설비효율을 극대화하는 전사적 생산보전 활동이다. 목표는 **고장 제로, 불량 제로, 재해 제로**이다. ([OEE.global][2])

### TPM의 기본 이념

* 예방보전
* 제로지향
* 전원참여
* 소집단 개선활동

### TPM 추진 단계

```text
도입 준비
     ↓
기본체계 구축
     ↓
자주보전 정착
     ↓
계획보전 추진
     ↓
개별개선 확대
     ↓
지속적 개선
```

### TPM의 8대 활동

| 활동        | 주요 내용       |
| --------- | ----------- |
| 개별개선      | 설비 손실 제거    |
| 자주보전      | 작업자 중심 일상관리 |
| 계획보전      | 예방·예측보전     |
| 품질보전      | 설비 기인 불량 예방 |
| 교육훈련      | 보전기술 향상     |
| 초기관리      | 신설비 조기 안정화  |
| 안전·환경     | 무재해 활동      |
| 사무간접부문 개선 | 지원부문 효율화    |

### TPM의 효과

* OEE 향상
* 고장 감소
* 품질 향상
* 생산성 향상
* 조직 참여문화 정착

## 자주보전

### 자주보전의 개요

자주보전(Autonomous Maintenance)은 작업자가 스스로 설비를 점검·청소·윤활하여 설비 이상을 조기에 발견하는 TPM의 핵심 활동이다.

### 자주보전 7단계

1. 초기 청소
2. 발생원 및 곤란개소 개선
3. 청소·급유 기준 설정
4. 총점검
5. 자주점검
6. 표준화
7. 자율관리

### 기대효과

* 설비 이상 조기 발견
* 작업자 설비 이해도 향상
* 보전부서 업무 경감

## 설비종합효율(OEE)

### OEE의 개요

OEE(Overall Equipment Effectiveness)는 설비가 생산에 얼마나 효과적으로 기여하는지를 나타내는 대표적인 종합 성과지표이다.

### 계산식

```text
OEE

=

가동률
×
성능효율
×
품질률
```

### 구성요소

| 항목   | 의미                 |
| ---- | ------------------ |
| 가동률  | 계획시간 대비 실제 가동시간    |
| 성능효율 | 이론 생산속도 대비 실제 생산속도 |
| 품질률  | 전체 생산량 대비 양품 비율    |

### 설비 6대 로스

| OEE 요소 | 손실             |
| ------ | -------------- |
| 가동률    | 고장, 준비·교체      |
| 성능효율   | 공회전·미세정지, 속도저하 |
| 품질률    | 초기 불량, 공정 불량   |

### 장치산업의 8대 로스

장치산업에서는 설비 6대 로스 외에 에너지 손실과 수율 손실 등을 포함한 8대 로스로 관리하는 경우가 많다.

## 설비 신뢰성 관리

### 신뢰성(Reliability)

일정 기간 동안 설비가 요구 기능을 수행할 확률

### 주요 지표

#### MTBF

평균 고장 간격

```text
MTBF

=

총 가동시간
──────────
고장 횟수
```

#### MTTR

평균 수리 시간

```text
MTTR

=

총 수리시간
──────────
고장 횟수
```

#### MTTF

평균 고장 시간

수리가 불가능한 설비의 평균 사용시간

#### 가용도(Availability)

```text
가용도

=

MTBF
────────────
MTBF + MTTR
```

> **MTBF는 클수록, MTTR은 작을수록 설비 신뢰성과 생산성이 향상된다.** ([Symestic][3])

## 욕조곡선(Bathtub Curve)

### 개요

설비의 고장률 변화를 시간에 따라 나타낸 곡선이다.

```text
고장률

↑
│＼
│ ＼__________
│            \
│             \
└────────────────→ 시간

초기고장   우발고장   마모고장
```

### 단계별 특징

| 구분    | 특징         | 관리방법       |
| ----- | ---------- | ---------- |
| 초기고장기 | 설치·제조 결함   | 초기 안정화     |
| 우발고장기 | 낮고 일정한 고장률 | 예방·상태기반 보전 |
| 마모고장기 | 노후화에 따른 증가 | 개량·교체      |

## RAMS

### 개요

RAMS는 설비의 종합적인 성능을 평가하는 대표적인 신뢰성 관리체계이다.

| 요소              | 의미  |
| --------------- | --- |
| Reliability     | 신뢰성 |
| Availability    | 가용성 |
| Maintainability | 보전성 |
| Safety          | 안전성 |

## 스마트 설비보전

### IoT 기반 설비 모니터링

센서를 활용하여 설비 상태를 실시간으로 수집·관리

### AI 기반 예지보전

설비 데이터를 분석하여 잔여수명(RUL)과 고장 시점을 예측

### 디지털 트윈

가상공장에서 설비 상태를 시뮬레이션하여 최적의 보전계획을 수립

### 스마트 TPM

기존 TPM에 IoT, AI, MES를 결합한 데이터 기반 설비관리 체계

## 설비보전관리의 종합

설비보전관리는 설비의 신뢰성과 생산성을 유지하기 위한 핵심 관리활동으로, 사후보전에서 TPM, RCM, CBM, PdM으로 지속적으로 발전하고 있다. 최근에는 AI와 디지털 트윈 기술을 활용한 스마트보전이 확산되면서 설비관리의 패러다임이 **고장 후 수리(BM) → 예방(PM) → 예측(PdM) → 자율보전(Self Maintenance)**으로 진화하고 있다. ([낟알 데이터][1])

### 범위 연계

| 연계 영역  | 주요 내용            |
| ------ | ---------------- |
| 생산계획수립 | 설비능력(CRP)        |
| 표준작업관리 | 자주점검, 표준작업       |
| 공정개선   | TPM, OEE, 카이젠    |
| 품질관리   | FMEA, SPC, 품질보전  |
| 스마트팩토리 | IoT, AI, 디지털 트윈  |
| 생산혁신기술 | Lean, TPM, 스마트보전 |

### 보완 권장 사항

현재 구성은 기술사 수준으로 충분하지만, 다음 내용을 추가하면 체계성이 더욱 향상된다.

* **RCM(Reliability Centered Maintenance)**: 설비 중요도에 따라 최적 보전전략을 선택하는 신뢰성 중심 보전
* **RUL(Remaining Useful Life)**: AI 기반 예지보전에서 잔여수명 예측 개념
* **TBM(Time Based Maintenance)**: 시간기준 보전으로 PM의 대표 실행방식
* **ISO 55000**: 자산관리 국제표준과 설비관리 연계
* **보전성(Maintainability)**과 **신뢰성(Reliability)**의 비교 정리

또한 **BM → PM(TBM/CBM) → TPM → PdM → AI 기반 자율보전**의 발전 흐름을 하나의 도식으로 정리하면 전체 설비보전 체계를 이해하기 쉽다.
