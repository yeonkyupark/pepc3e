## 공정개선

공정개선(Process Improvement)은 생산성과 품질을 향상시키고 원가를 절감하기 위하여 기존 공정의 낭비와 비효율을 제거하는 활동이다. 현대 제조기업은 단순한 생산능력 확대보다 공정혁신을 통한 경쟁력 확보를 중요시하며, Lean, TPS, Six Sigma, TOC 등의 개선기법을 활용하여 지속적인 성과향상을 추구하고 있다.

### 공정개선의 개요

#### 공정개선의 정의

공정개선이란 생산공정의 품질, 원가, 납기, 안전 및 유연성을 향상시키기 위하여 작업방법과 공정구조를 지속적으로 개선하는 활동이다.

#### 공정개선의 목적

* 생산성 향상
* 원가 절감
* 품질 향상
* 납기 단축
* 안전 확보
* 고객가치 증대

#### 공정개선의 기본 방향

```text
품질 향상
    ↑
공정개선
    ↓
원가 절감
    ↓
납기 단축
```

궁극적으로 QCDFS 경쟁우선순위 달성을 목표로 한다.

## 공정개선의 접근방법

### 개선 활동의 수준

공정개선은 개선 범위와 혁신 수준에 따라 점진적 개선과 혁신적 개선으로 구분할 수 있다.

| 구분                     | 특징         |
| ---------------------- | ---------- |
| Kaizen                 | 지속적·점진적 개선 |
| Innovation             | 급진적·혁신적 개선 |
| BPR                    | 프로세스 재설계   |
| Digital Transformation | 디지털 기반 혁신  |

### 개선활동의 순환구조

```text
현상파악
    ↓
원인분석
    ↓
개선안 도출
    ↓
실행
    ↓
효과확인
    ↓
표준화
```

### 개선의 기본원칙

#### 3정 5S

공정개선의 기본은 작업장 정돈과 표준화이다.

##### 3정

* 정품
* 정량
* 정위치

##### 5S

| 구분  | 내용           |
| --- | ------------ |
| 정리  | 불필요한 것 제거    |
| 정돈  | 필요한 것 정위치 보관 |
| 청소  | 작업장 청결 유지    |
| 청결  | 정리·정돈·청소의 유지 |
| 습관화 | 생활화 및 정착     |

## 공정분석

### 공정분석의 개요

공정분석은 공정의 흐름을 체계적으로 분석하여 개선기회를 발견하는 기법이다.

### 공정분석의 대상

* 작업
* 운반
* 검사
* 대기
* 저장

### 공정분석 기호

| 기호 | 의미                 |
| -- | ------------------ |
| ○  | 작업(Operation)      |
| →  | 운반(Transportation) |
| □  | 검사(Inspection)     |
| D  | 대기(Delay)          |
| ▽  | 저장(Storage)        |

### 공정분석의 목적

* 낭비 발견
* 병목 제거
* 리드타임 단축
* 생산성 향상

!!! note "범위 연계"

```
공정분석도(Process Chart)와 유동분석은 「프로세스 분석과 개선」에서 상세히 설명한다.
```

## ECRS 개선기법

### 개요

ECRS는 작업개선의 가장 기본적인 원칙이다.

### ECRS 구성

| 구분        | 의미  |
| --------- | --- |
| Eliminate | 제거  |
| Combine   | 결합  |
| Rearrange | 재배치 |
| Simplify  | 단순화 |

### 적용 순서

개선 시에는 반드시 E → C → R → S 순서로 검토한다.

### 적용 사례

| 문제      | 개선방안           |
| ------- | -------------- |
| 불필요한 검사 | 제거(Eliminate)  |
| 중복 작업   | 결합(Combine)    |
| 동선 비효율  | 재배치(Rearrange) |
| 복잡한 작업  | 단순화(Simplify)  |

## TPS 기반 공정개선

### TPS의 개선철학

TPS는 낭비 제거를 통해 고객가치를 극대화하는 생산방식이다.

### 3M

공정개선은 3M 제거를 중심으로 수행된다.

| 구분   | 의미  |
| ---- | --- |
| Muda | 낭비  |
| Mura | 불균형 |
| Muri | 무리  |

### 7대 낭비

| 구분   | 내용       |
| ---- | -------- |
| 과잉생산 | 필요 이상 생산 |
| 대기   | 작업 대기    |
| 운반   | 불필요한 이동  |
| 가공   | 불필요한 공정  |
| 재고   | 과다 재고    |
| 동작   | 불필요한 동작  |
| 불량   | 재작업 및 폐기 |

### 개선 방향

```text
7대 낭비 제거
        ↓
리드타임 단축
        ↓
재고 감소
        ↓
생산성 향상
```

## 가치흐름분석(VSM)

### 개요

VSM(Value Stream Mapping)은 가치창출 과정 전체를 시각적으로 분석하는 기법이다.

### 목적

* 가치창출 활동 식별
* 비부가가치 활동 제거
* 리드타임 단축
* 흐름 개선

### 분석 절차

```text
현재상태 분석
        ↓
낭비 파악
        ↓
미래상태 설계
        ↓
개선 실행
```

### 기대효과

* 전체 최적화
* 공정 간 연계성 강화
* 재고 감소

## 제약이론(TOC)

### 개요

TOC(Theory of Constraints)는 시스템의 성과를 제한하는 병목을 중심으로 개선하는 관리기법이다.

### 기본 개념

시스템의 성과는 가장 약한 고리에 의해 결정된다.

### 집중개선 5단계

```text
병목 발견
    ↓
병목 활용
    ↓
병목 지원
    ↓
제약 제거
    ↓
새로운 병목 탐색
```

### Drum-Buffer-Rope

| 요소     | 역할      |
| ------ | ------- |
| Drum   | 생산속도 결정 |
| Buffer | 변동성 흡수  |
| Rope   | 생산동기화   |

### 기대효과

* 생산량 증가
* 재공품 감소
* 납기 준수율 향상

## Six Sigma

### 개요

Six Sigma는 데이터 기반의 품질혁신 및 공정개선 기법이다.

### 목표

불량을 백만 기회당 3.4개 수준으로 감소시키는 것[^1]

### DMAIC

```text
Define
    ↓
Measure
    ↓
Analyze
    ↓
Improve
    ↓
Control
```

### 주요 도구

* 관리도
* 파레토도
* 특성요인도
* 회귀분석
* 실험계획법

[^1]: 장기 공정능력을 고려한 Six Sigma 수준.

!!! note "범위 연계"

```
통계적 품질관리 기법은 「품질관리」에서 상세히 설명한다.
```

## BPR

### 개요

BPR(Business Process Reengineering)은 기존 업무 프로세스를 근본적으로 재설계하여 성과를 획기적으로 향상시키는 혁신기법이다.

### 핵심 특징

* 근본적 재고
* 급진적 변화
* 프로세스 중심
* IT 활용

### 기대효과

* 업무시간 단축
* 비용 절감
* 고객만족 향상

### 한계

* 조직 저항
* 변화관리 필요
* 실패 위험 존재

## 공정개선의 종합

공정개선은 생산성 향상과 경쟁력 확보를 위한 핵심 관리활동이다. 공정분석, ECRS, TPS, TOC, Six Sigma 및 BPR 등의 다양한 개선기법을 활용하여 낭비를 제거하고 병목을 해소함으로써 품질·원가·납기 경쟁력을 향상시킬 수 있다. 최근에는 스마트팩토리와 AI 기술이 결합되면서 데이터 기반의 실시간 공정개선이 가능해지고 있으며, ESG와 지속가능성까지 고려한 통합적 개선체계로 발전하고 있다.

### 범위 연계

| 연계 영역       | 주요 내용               |
| ----------- | ------------------- |
| 작업방법 및 시간연구 | 동작개선, ECRS          |
| 프로세스 분석과 개선 | 공정분석도, 유동분석         |
| 표준작업관리      | 표준화, SDCA           |
| 품질관리        | Six Sigma, 통계적 품질관리 |
| 생산혁신기술      | TPM, Lean, 혁신활동     |
| 스마트팩토리      | 디지털 트윈, AI 기반 최적화   |

### 보완 권장 사항

현재 공정개선 단원에서 추가하면 좋은 핵심 내용은 다음과 같다.

* IE(Industrial Engineering)와 공정개선의 관계
* VE(Value Engineering)와 공정개선의 차이
* Kaizen과 혁신(Innovation)의 비교
* Poka-Yoke(실수방지) 기법
* 공정개선 성과지표(KPI)

  * 생산성
  * OEE
  * 리드타임
  * 재공품(WIP)
  * 불량률
* Lean, TOC, Six Sigma 비교 체계

특히 공장관리기술사 관점에서는 **Lean-TPS, TOC, Six Sigma를 비교하는 통합 프레임워크**를 추가하면 공정개선 단원의 완성도가 크게 향상된다.
