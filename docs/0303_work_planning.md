## 작업계획수립

작업계획수립(Work Planning)은 생산계획과 공정설계를 기반으로 작업자, 설비, 자재 및 작업방법을 구체적으로 계획하여 생산활동을 실행 가능한 수준으로 구체화하는 활동이다. 생산계획이 무엇을 얼마나 생산할 것인지를 결정한다면, 작업계획은 누가, 언제, 어디서, 어떻게 작업할 것인지를 결정하는 단계이다.

### 작업계획의 개요

#### 작업계획의 정의

작업계획이란 생산목표를 달성하기 위하여 작업순서, 작업방법, 작업자 배치 및 자원 활용방안을 구체적으로 수립하는 활동이다.

#### 작업계획의 목적

* 생산활동의 표준화
* 작업효율 향상
* 생산성 증대
* 납기 준수
* 품질 확보
* 안전성 향상

#### 작업계획의 위치

```text
수요예측
    ↓
생산계획
    ↓
공정설계
    ↓
작업계획
    ↓
작업실행
    ↓
진도관리
```

## 작업계획의 구성요소

### 작업계획의 4대 요소

작업계획은 사람, 설비, 자재 및 방법을 중심으로 수립된다.

| 요소       | 주요 내용           |
| -------- | --------------- |
| Man      | 작업자, 기능수준, 인원배치 |
| Machine  | 설비능력, 설비배치      |
| Material | 자재 및 공구 준비      |
| Method   | 작업방법, 작업순서      |

### 작업계획 수립 절차

```text
작업분석
    ↓
작업방법 결정
    ↓
작업순서 결정
    ↓
작업자 배치
    ↓
설비 및 자재 준비
    ↓
작업표준 작성
```

## 작업분해구조(WBS)

### WBS의 개요

WBS(Work Breakdown Structure)는 전체 작업을 관리 가능한 단위로 계층적으로 분해하는 기법이다.

### 목적

* 작업범위 명확화
* 일정계획 수립
* 자원배분 지원
* 진도관리 용이성 확보

### 구조

```text
프로젝트
    ├─ 대공정
    │     ├─ 중공정
    │     │     ├─ 작업
    │     │     └─ 작업
    │     └─ 중공정
    └─ 대공정
```

### 기대효과

* 작업 누락 방지
* 책임 명확화
* 계획 정확성 향상

## 작업순서 계획

### 작업순서의 개요

작업순서(Routing)는 원자재가 완제품이 되기까지 거치는 공정의 순서를 결정하는 활동이다.

### 작업순서 결정요인

* 제품 구조
* 공정 특성
* 설비 능력
* 품질 요구사항
* 생산량

### 작업순서표

작업순서표는 공정 흐름과 작업 내용을 문서화한 것이다.

| 공정번호 | 작업내용 | 사용설비 | 표준시간 |
| ---- | ---- | ---- | ---- |
| 10   | 절단   | 절단기  | 5분   |
| 20   | 가공   | CNC  | 10분  |
| 30   | 조립   | 조립라인 | 8분   |
| 40   | 검사   | 검사설비 | 3분   |

## 작업방법 계획

### 작업방법의 개요

작업방법(Method)은 작업을 수행하는 절차와 방식을 의미한다.

### 작업방법 설계 원칙

#### ECRS 원칙

| 원칙        | 내용  |
| --------- | --- |
| Eliminate | 제거  |
| Combine   | 결합  |
| Rearrange | 재배치 |
| Simplify  | 단순화 |

#### 경제동작 원칙

작업자의 동작을 최소화하여 작업효율을 향상시키는 원칙이다.[^1]

##### 기본 방향

* 불필요한 동작 제거
* 양손 동시 활용
* 동작거리 최소화
* 작업장 정리정돈

[^1]: Gilbreth 부부가 제안한 동작연구(Motion Study)의 기본 원리.

!!! note "관련 단원"

```
경제동작 원칙과 동작연구는 「작업방법 및 시간연구」에서 상세히 설명한다.
```

## 작업자 배치계획

### 작업자 배치의 개요

작업자의 능력과 작업 특성을 고려하여 적절한 인력을 배치하는 활동이다.

### 배치 원칙

* 적재적소 배치
* 작업부하 균형
* 다기능공 활용
* 안전성 확보

### 다기능공(Multi-skilled Worker)

여러 공정을 수행할 수 있도록 교육된 작업자

#### 기대효과

* 생산 유연성 향상
* 인력 활용도 향상
* 병목 대응력 향상

!!! note "관련 단원"

```
다기능공 육성과 직무순환은 「인력과 직무설계」에서 상세히 설명한다.
```

## 작업공수 계획

### 작업공수의 개요

공수(工數)는 작업 수행에 필요한 노동량을 의미한다.

### 공수 산정

```text
총 공수

=

생산수량
×
표준시간
```

### 공수 활용

* 인원계획
* 생산능력 산정
* 원가계산
* 일정계획

### 여유율

실제 작업에서는 피로, 작업지연 및 개인차를 고려하여 여유율을 반영한다.

```text
표준시간

=

정미시간
+
여유시간
```

## 라인 밸런싱(Line Balancing)

### 개요

라인 밸런싱은 각 공정의 작업부하를 균형 있게 배분하여 생산효율을 향상시키는 기법이다.

### 목적

* 병목공정 제거
* 대기시간 감소
* 설비가동률 향상
* 생산성 향상

### 주요 지표

#### 사이클타임(Cycle Time)

연속된 제품 생산 간격

#### 택트타임(Takt Time)

고객수요 기준 생산속도

```text
택트타임

=

가용생산시간
────────────
고객수요량
```

#### 밸런스 효율

```text
라인효율

=

총 작업시간
─────────────────
공정수 × 사이클타임
```

### 기대효과

* 공정 간 균형 확보
* 생산능력 향상
* 납기 준수

!!! note "관련 단원"

```
택트타임과 셀 생산방식은 「생산구조와 방식(Layout)」에서 상세히 설명한다.
```

## 작업지시와 작업통제

### 작업지시

생산계획을 작업현장에 전달하는 활동

#### 주요 내용

* 생산품목
* 생산수량
* 작업순서
* 납기
* 품질기준

### 작업통제

계획 대비 실적을 비교하여 차이를 수정하는 활동

#### 통제 절차

```text
계획수립
    ↓
작업실행
    ↓
실적측정
    ↓
차이분석
    ↓
시정조치
```

### 작업표준의 개요

* 표준작업순서
* 표준시간
* 표준재공량

### 효과

* 품질 안정화
* 생산성 향상
* 작업자 교육
* 개선활동 기반 구축

## 디지털 작업계획

### MES 기반 작업계획

MES는 작업지시, 실적수집 및 작업통제를 실시간으로 수행한다.

### APS 연계

APS는 생산계획과 작업계획을 연계하여 최적 작업일정을 생성한다.

### 디지털 작업표준

전자 작업표준서를 활용하여 작업정보를 실시간 제공하는 방식

### AI 기반 작업계획

인공지능을 활용하여 작업배치, 일정계획 및 병목예측을 수행하는 기술

## 최근 동향

### 스마트 작업관리

IoT 및 MES를 활용한 실시간 작업관리

### 자율생산(Self-Optimizing Production)

현장 데이터를 기반으로 작업조건을 자동 최적화하는 생산방식

### 디지털 작업지시

종이 작업지시서를 대체하는 전자 작업지시 체계

### 인간 중심 생산시스템

자동화와 작업자 역량을 조화시키는 Human-Centric Manufacturing이 강조되고 있다.

## 작업계획수립의 종합

작업계획은 생산계획과 현장 실행을 연결하는 핵심 관리기능이다. 작업순서, 작업방법, 작업자 배치 및 작업표준을 체계적으로 계획함으로써 생산성, 품질 및 납기 경쟁력을 확보할 수 있다. 최근에는 MES, APS, AI 및 스마트팩토리 기술을 활용하여 실시간 작업계획과 자율적 생산운영이 가능해지고 있으며, 이는 제조 경쟁력 향상의 핵심 기반이 되고 있다.

### 범위 연계

| 연계 영역       | 주요 내용              |
| ----------- | ------------------ |
| 생산계획수립      | MPS, MRP, CRP      |
| 공정설계        | 작업순서 및 공정구성        |
| 표준작업관리      | 작업표준 및 표준시간        |
| 작업방법 및 시간연구 | 동작연구, 시간연구         |
| 공정개선        | ECRS, 병목개선         |
| 스마트팩토리      | MES, APS, 디지털 작업관리 |

### 퇴고 의견

현재 구성은 전반적으로 양호하지만, 기술사 및 생산관리 체계 관점에서는 **WBS는 다소 프로젝트 관리(PM) 성격이 강하므로 비중을 줄이고**, 대신 아래 내용을 보강하는 것을 권장한다.

* Routing(공정경로계획)
* Loading(부하계획)
* Scheduling(일정계획)
* Dispatching(작업배정)
* Follow-up(진도확인)

즉 전통적인 작업계획 체계인

```text
Routing
 ↓
Loading
 ↓
Scheduling
 ↓
Dispatching
 ↓
Follow-up
```

를 중심축으로 구성하면 생산관리 교과서 및 공장관리기술사 출제체계와 더욱 일치한다.

특히 **Loading(부하계획)** 과 **Dispatching(작업배정)** 은 현재 초안에서 상대적으로 부족한 부분으로 보인다. 이를 보강하면 작업계획수립 단원의 완성도가 한 단계 높아질 것이다.
