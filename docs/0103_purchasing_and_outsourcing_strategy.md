## 구매전략
구매전략은 구매의 정의, 구매관리, 구매가격, 할인방식, 구매방식 유형으로 구성한다.

### 구매 정의
구매 정의는 구매와 조달의 개념 구분, 전통적 시각과 현대적 시각의 차이로 구성한다.

구매 정의
: - 구매와 조달의 개념 구분
- 전통적 시각과 현대적 시각의 차이

### 구매관리
구매관리는 구매관리의 목적(5R)으로 구성한다.

구매관리
: - 구매관리의 목적(5R)

### 구매가격
구매가격은 가격 결정방식과 가격 유형으로 구성한다.

구매가격
: - 가격 결정방식(비용중심, 구매자중심, 경쟁자중심)
- 가격 유형(시중가격, 개정가격, 정가가격, 협정가격, 교섭가격)

### 할인 방식
할인 방식은 현금할인 방식과 수량할인 방식으로 구성한다.

할인 방식
: - 현금할인 방식
- 수량할인 방식

### 구매방식 유형
구매방식 유형은 집중구매와 분산구매로 구성한다.

구매방식 유형
: - 집중구매와 분산구매의 개념과 장단점

## 생산 및 외주 의사결정
생산 및 외주 의사결정은 자가생산과 외주생산, 수직적 통합, 오프쇼어링과 리쇼어링으로 구성한다.

### 자가생산과 외주생산
자가생산과 외주생산은 생산방식 결정의 기본 대안이다.

자가생산과 외주생산
: - 자가생산은 품질 및 기술력 내재화
- 외주생산은 비용 절감 및 유연성 확보
- 자가생산과 외주생산의 장단점 비교

### 수직적 통합
수직적 통합은 전방통합과 후방통합으로 구분한다.

수직적 통합
: - 전방통합은 소비자 방향으로 사업영역 확대
- 후방통합은 공급자 방향으로 사업영역 확대

### 오프쇼어링과 리쇼어링
오프쇼어링과 리쇼어링은 글로벌 생산입지 전략의 양축이다.

오프쇼어링과 리쇼어링
: - 오프쇼어링은 해외로의 생산시설 이전
- 리쇼어링은 해외 생산시설의 자국 회귀

**(추가 권장)** 자가생산·외주·수직적통합·오프쇼어링 간 관계도

## 외주전략
외주전략은 외주전략 목적과 아웃소싱으로 구성한다.

### 외주전략 목적
외주전략 목적은 핵심역량 강화, 위험분산, 경영효율화, 비용절감으로 구성한다.

외주전략 목적
: - 핵심역량 강화
- 위험 분산
- 경영 효율화
- 비용 절감

### 아웃소싱
아웃소싱은 외주전략의 실행 수단이다.

아웃소싱
: - 아웃소싱의 목적(비용절감, 핵심역량 집중, 전문성 확보, 유연성 증가)
- 아웃소싱의 유형(BPO, IT 아웃소싱, 제조 아웃소싱)
- 아웃소싱의 장단점
