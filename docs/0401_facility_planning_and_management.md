## 설비계획 및 관리

설비계획 및 관리(Facility Planning and Management)는 생산목표를 달성하기 위해 설비를 계획·선정·도입·운영·개선하는 전 과정을 관리하는 활동이다. 설비는 생산능력과 품질수준을 결정하는 핵심 생산자원으로, 적정한 설비계획과 체계적인 운영관리는 생산성 향상과 원가절감, 품질안정 및 경쟁력 확보의 기반이 된다.

### 설비계획 및 관리의 개요

#### 설비의 정의

설비(Facility)란 제품 또는 서비스를 생산하기 위해 사용하는 기계, 장치, 건물, 운반설비 및 부대시설을 말한다.

#### 설비관리의 목적

* 생산능력 확보
* 품질 안정화
* 생산성 향상
* 원가 절감
* 설비수명 연장
* 안전성 확보

#### 설비관리의 범위

```text
설비계획
    ↓
설비선정
    ↓
설비도입
    ↓
설비운영
    ↓
설비개선
    ↓
설비폐기
```

## 설비계획

### 설비계획의 개요

설비계획은 생산전략과 생산계획을 실현하기 위하여 필요한 설비의 종류, 규모 및 능력을 결정하는 활동이다.

#### 설비계획의 목적

* 생산목표 달성
* 생산능력 확보
* 투자효율 향상
* 미래 수요 대응
* 자동화 기반 구축

#### 설비계획 절차

```text
생산목표 설정
      ↓
수요예측
      ↓
생산능력 산정
      ↓
설비규모 결정
      ↓
설비선정
      ↓
경제성 평가
      ↓
설비도입
```

### 설비계획의 고려요소

#### 생산 측면

* 생산량
* 제품 특성
* 공정 특성
* 품질 수준

#### 경제성 측면

* 초기 투자비
* 운영비
* 유지보전비
* 회수기간

#### 기술성 측면

* 자동화 수준
* 신뢰성
* 보전성
* 확장성

## 설비능력계획

### 생산능력(Capacity)

생산능력이란 일정 기간 동안 생산할 수 있는 최대 생산량을 의미한다.

### 생산능력의 구분

| 구분    | 의미                     |
| ----- | ---------------------- |
| 설계능력  | 설비가 설계상 달성 가능한 최대 생산능력 |
| 유효능력  | 현실적으로 확보 가능한 생산능력      |
| 실제생산량 | 실제 생산된 생산실적            |

### 능력계획 전략

| 전략          | 특징          | 적용 상황   |
| ----------- | ----------- | ------- |
| 선행전략(Lead)  | 수요 증가 이전 증설 | 성장시장    |
| 추종전략(Lag)   | 수요 증가 후 증설  | 안정시장    |
| 매칭전략(Match) | 단계적 증설      | 수요 변동시장 |

!!! note "범위 연계"

```
생산능력계획(CRP)의 상세 내용은 「생산계획수립」에서 설명한다.
```

## 설비선정

### 설비선정의 개요

설비선정은 생산목표와 경제성을 고려하여 최적의 설비를 결정하는 활동이다.

### 선정 기준

#### 기술성

* 성능
* 정밀도
* 신뢰성
* 자동화 수준

#### 경제성

* 취득비
* 운영비
* 유지보전비
* 수명주기비용(LCC)

#### 운영성

* 작업성
* 안전성
* 보전성
* 확장성

### 설비 선정 절차

```text
설비요구사항 분석
        ↓
후보설비 선정
        ↓
기술성 평가
        ↓
경제성 평가
        ↓
최종 선정
```

## 경제성 분석

### 경제성 분석의 목적

설비 투자안의 경제성을 평가하여 최적의 투자 의사결정을 지원한다.

### 주요 평가기법

| 기법            | 특징            |
| ------------- | ------------- |
| 손익분기점(BEP) 분석 | 생산량 기준 경제성 비교 |
| 투자회수기간(PP)    | 투자금 회수기간 평가   |
| 순현재가치(NPV)    | 현금흐름의 현재가치 평가 |
| 내부수익률(IRR)    | 투자수익률 평가      |

!!! note "범위 연계"

```
경제성 분석은 「사업성 분석」에서 상세히 설명한다.
```

## 설비배치

### 설비배치의 목적

설비배치는 생산흐름을 최적화하여 운반거리와 대기시간을 최소화하는 활동이다.

### 설비배치의 원칙

* 물류흐름 최소화
* 공간 활용 극대화
* 작업 안전 확보
* 생산 유연성 확보

### 설비배치 유형

| 유형     | 특징               |
| ------ | ---------------- |
| 위치고정형  | 제품이 고정되고 작업자가 이동 |
| 공정별 배치 | 기능별 설비 배치        |
| 제품별 배치 | 공정순으로 설비 배치      |
| 셀형 배치  | 제품군 중심 배치        |

!!! note "범위 연계"

```
설비배치(Layout)는 「생산구조와 방식(Layout)」에서 상세히 설명한다.
```

## 설비관리

### 설비관리의 개요

설비관리는 설비의 성능을 유지하고 생산성을 극대화하기 위한 운영관리 활동이다.

### 관리 대상

* 생산설비
* 운반설비
* 검사설비
* 유틸리티 설비
* 안전설비

### 주요 관리활동

* 설비점검
* 성능관리
* 이력관리
* 개선활동
* 교체계획

## 설비효율 관리

### 설비종합효율(OEE)

설비의 생산성을 종합적으로 평가하는 대표적인 성과지표이다.

#### 계산식

```text
OEE

=

가동률
×
성능효율
×
양품률
```

#### 구성요소

| 요소   | 의미              |
| ---- | --------------- |
| 가동률  | 계획 대비 실제 가동시간   |
| 성능효율 | 설계속도 대비 실제 생산속도 |
| 양품률  | 전체 생산량 대비 양품 비율 |

### 설비 손실

#### 설비 6대 로스

| 구분       | 내용       |
| -------- | -------- |
| 고장손실     | 설비 고장    |
| 준비교체손실   | 셋업 및 교체  |
| 공회전·미세정지 | 일시 정지    |
| 속도저하     | 설계속도 미달  |
| 불량손실     | 공정 불량    |
| 초기수율손실   | 생산 초기 불량 |

## 설비 신뢰성 관리

### 신뢰성(Reliability)

일정 기간 동안 설비가 요구 기능을 수행할 확률

### 주요 지표

#### MTBF

평균고장간격

```text
MTBF

=

총 가동시간
──────────
고장횟수
```

#### MTTR

평균수리시간

```text
MTTR

=

총 수리시간
──────────
고장횟수
```

#### 가용도(Availability)

```text
가용도

=

MTBF
────────────
MTBF + MTTR
```

### 욕조곡선(Bathtub Curve)

설비의 수명주기별 고장률 변화를 나타내는 대표적인 신뢰성 곡선이다.

| 구간    | 특징       | 관리방안   |
| ----- | -------- | ------ |
| 초기고장기 | 제조·설치 결함 | 초기 안정화 |
| 우발고장기 | 고장률 일정   | 예방보전   |
| 마모고장기 | 노후화 진행   | 개량·교체  |

## 설비관리의 최근 동향

### Life Cycle Cost(LCC)

도입부터 폐기까지 발생하는 총비용을 고려하는 설비관리 방식

### 스마트 설비관리

IoT와 AI를 활용하여 설비 상태를 실시간으로 모니터링하고 최적화하는 관리체계

### 디지털 트윈

설비의 가상모델을 구축하여 성능과 이상상태를 예측하는 기술

### 자산관리(ISO 55000)

설비를 기업의 전략적 자산으로 관리하는 국제표준

## 설비계획 및 관리의 종합

설비계획 및 관리는 생산전략을 실현하기 위한 핵심 기반으로, 적정한 설비능력 확보와 경제적인 설비투자를 통해 기업 경쟁력을 결정한다. 또한 설비운영 단계에서는 OEE와 신뢰성 지표를 활용하여 설비효율을 지속적으로 향상시키며, 최근에는 스마트팩토리 기술과 결합하여 데이터 기반의 지능형 설비관리 체계로 발전하고 있다.

### 범위 연계

| 연계 영역        | 주요 내용           |
| ------------ | --------------- |
| 생산구조와 방식     | 설비배치(Layout)    |
| 생산계획수립       | 생산능력계획(CRP)     |
| 설비보전관리       | TPM, 예방보전, OEE  |
| 시설 및 유틸리티 관리 | 공장 기반시설 운영      |
| 사업성 분석       | 경제성 분석, 투자평가    |
| 스마트팩토리       | IoT, AI, 디지털 트윈 |



이렇게 역할을 명확히 분리하면 두 단원이 중복 없이 자연스럽게 연결되며, 전체 교재의 논리적 흐름도 더욱 좋아진다.
