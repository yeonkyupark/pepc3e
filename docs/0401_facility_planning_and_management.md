## 설비계획 및 관리

설비계획 및 관리(Facility Planning and Management)는 기업의 생산목표를 달성하기 위하여 설비를 계획·도입·운영·보전·폐기하는 전 과정을 관리하는 활동이다. 설비는 생산능력과 품질수준을 결정하는 핵심 자산이며, 최근에는 스마트팩토리와 예지보전 기술의 발전으로 설비관리의 중요성이 더욱 커지고 있다.

### 설비관리의 개요

#### 설비의 정의

설비(Facility)란 생산활동을 수행하기 위하여 사용되는 기계, 장치, 건물, 운반설비 및 유틸리티 설비를 의미한다.

#### 설비관리의 목적

* 생산능력 확보
* 품질 안정화
* 설비효율 향상
* 원가 절감
* 안전성 확보
* 설비수명 연장

#### 설비 생애주기(Life Cycle)

설비는 계획에서 폐기에 이르는 생애주기 관점에서 관리되어야 한다.

```text
설비계획
    ↓
설비선정
    ↓
설비도입
    ↓
운영
    ↓
보전
    ↓
개선
    ↓
폐기 및 대체
```

## 설비계획

### 설비계획의 개요

설비계획은 생산전략과 생산능력계획을 바탕으로 필요한 설비를 결정하는 활동이다.

### 설비계획의 목적

* 생산능력 확보
* 투자효율 향상
* 미래 수요 대응
* 기술경쟁력 확보

### 설비계획 절차

```text
생산목표 설정
    ↓
수요예측
    ↓
생산능력 분석
    ↓
설비규모 결정
    ↓
설비선정
    ↓
경제성 평가
    ↓
도입 및 운영
```

### 설비계획 시 고려요소

#### 생산측면

* 생산량
* 제품특성
* 공정특성
* 품질수준

#### 경제성 측면

* 초기투자비
* 운영비
* 유지보전비
* 회수기간

#### 기술적 측면

* 자동화 수준
* 확장성
* 신뢰성
* 표준화 수준

## 생산능력계획

### 생산능력의 개요

생산능력(Capacity)은 일정 기간 동안 생산할 수 있는 최대 생산량을 의미한다.

### 생산능력의 구분

| 구분    | 의미           |
| ----- | ------------ |
| 설계능력  | 설계상 최대 생산능력  |
| 유효능력  | 실제 운영 가능한 능력 |
| 실제생산량 | 실제 생산실적      |

### 생산능력 관련 지표

#### 설비가동률

```text
가동률

=

실제생산량
──────────
유효능력
×100
```

#### 설비이용률

```text
이용률

=

실제생산량
──────────
설계능력
×100
```

### 능력계획 전략

| 전략          | 특징              |
| ----------- | --------------- |
| 선행전략(Lead)  | 수요 증가 이전 선제적 증설 |
| 추종전략(Lag)   | 수요 증가 후 증설      |
| 매칭전략(Match) | 단계적 증설          |

!!! note "범위 연계"

```
생산능력계획(CRP)은 「생산계획수립」에서 상세히 설명한다.
```

## 설비배치와 설비관리

### 설비배치의 중요성

설비배치는 생산성, 물류효율 및 작업안전에 직접적인 영향을 미친다.

### 설비배치 목표

* 운반거리 최소화
* 물류흐름 최적화
* 공간효율 향상
* 안전성 확보

### 배치유형

| 유형     | 특징          |
| ------ | ----------- |
| 위치고정형  | 제품이 이동하지 않음 |
| 공정별 배치 | 기능 중심 배치    |
| 제품별 배치 | 흐름 중심 배치    |
| 셀형 배치  | 유연성 중심 배치   |

## 설비보전관리

### 설비보전의 개요

설비보전(Maintenance)은 설비의 기능을 유지하거나 복원하기 위한 관리활동이다.

### 설비보전의 목적

* 고장 예방
* 설비수명 연장
* 생산중단 방지
* 품질 안정화
* 안전 확보

### 보전활동의 기본구조

```text
설비고장 예방
      ↓
가동률 향상
      ↓
생산성 향상
      ↓
원가 절감
```

## 설비보전 방식

### 사후보전(BM)

Breakdown Maintenance

고장 발생 후 수리하는 방식

#### 장점

* 초기관리비 적음
* 관리가 단순

#### 단점

* 돌발고장 발생
* 생산손실 위험

### 예방보전(PM)

Preventive Maintenance

고장 발생 이전에 정기적으로 점검하는 방식

#### 장점

* 고장 예방
* 설비 신뢰성 향상

#### 단점

* 과잉보전 가능성
* 보전비 증가

### 개량보전(CM)

Corrective Maintenance

설비를 개선하여 고장을 감소시키는 활동

### 보전예방(MP)

Maintenance Prevention

설비 설계단계부터 보전성을 고려하는 활동

### 예지보전(PdM)

Predictive Maintenance

센서와 데이터를 활용하여 고장을 예측하는 방식

#### 특징

* 상태기반 보전
* AI 활용 가능
* 스마트보전 핵심기술

## TPM(Total Productive Maintenance)

### TPM의 개요

TPM은 생산부문과 보전부문이 함께 설비효율을 극대화하는 전사적 생산보전 활동이다.[^1]

### TPM의 목표

* 고장 제로
* 불량 제로
* 재해 제로

### TPM의 8대 추진기둥

| 구분        | 내용          |
| --------- | ----------- |
| 개별개선      | 설비효율 향상     |
| 자주보전      | 작업자 중심 보전   |
| 계획보전      | 체계적 보전활동    |
| 품질보전      | 품질결함 예방     |
| 교육훈련      | 기능향상        |
| 초기관리      | 초기설비 안정화    |
| 안전환경      | 안전 및 환경관리   |
| 사무간접부문 개선 | 지원부문 생산성 향상 |

[^1]: TPM은 일본의 생산보전협회(JIPM)를 중심으로 발전한 전사적 설비관리 활동이다.

### 자주보전

운영자가 직접 수행하는 설비관리 활동

#### 기대효과

* 설비이상 조기 발견
* 설비애착 형성
* 보전부문 부담 감소

## 설비종합효율(OEE)

### OEE의 개요

OEE(Overall Equipment Effectiveness)는 설비효율을 종합적으로 평가하는 대표적인 지표이다.

### OEE 구성요소

| 요소   | 의미            |
| ---- | ------------- |
| 가동률  | 설비가 실제 가동된 비율 |
| 성능효율 | 설계속도 대비 생산속도  |
| 양품률  | 생산품 중 양품 비율   |

### 계산식

```text
OEE

=

가동률
×
성능효율
×
양품률
```

### 6대 로스

| 구분     | 내용      |
| ------ | ------- |
| 고장손실   | 설비고장    |
| 준비교체손실 | 셋업 및 교체 |
| 공회전손실  | 일시정지    |
| 속도저하손실 | 성능 저하   |
| 불량손실   | 품질불량    |
| 초기수율손실 | 초기 불안정  |

## 설비진단과 신뢰성 관리

### 설비진단

설비 상태를 점검하여 이상 여부를 판단하는 활동

#### 진단기법

* 진동분석
* 열화상 분석
* 윤활유 분석
* 초음파 진단

### 신뢰성(Reliability)

설비가 일정 조건에서 요구 기능을 수행할 확률

### 주요 지표

#### MTBF

평균고장간격

```text
MTBF

=

총 가동시간
──────────
고장횟수
```

#### MTTR

평균수리시간

```text
MTTR

=

총 수리시간
──────────
고장횟수
```

#### 가용도(Availability)

```text
가용도

=

MTBF
────────────
MTBF + MTTR
```

## 설비계획 및 관리의 종합

설비계획 및 관리는 생산시스템의 경쟁력을 결정하는 핵심 관리영역이다. 설비계획은 생산전략을 실현하기 위한 물적 기반을 구축하는 활동이며, 설비보전은 구축된 설비의 성능을 지속적으로 유지·향상시키는 활동이다. 최근에는 TPM, OEE, 신뢰성 공학 및 AI 기반 예지보전 기술이 결합되면서 설비관리는 단순 유지관리에서 데이터 기반의 전략적 자산관리 체계로 발전하고 있다.

### 범위 연계

| 연계 영역     | 주요 내용             |
| --------- | ----------------- |
| 생산계획수립    | 생산능력계획(CRP)       |
| 생산구조와 방식  | 설비배치(Layout)      |
| 표준작업관리    | 설비점검표준            |
| 공정개선      | TPM, OEE 개선       |
| 환경 및 안전관리 | 설비안전, 위험성 평가      |
| 스마트팩토리    | IoT, 디지털 트윈, 예지보전 |

### 보완 권장 사항

설비계획과 설비보전을 하나의 단원으로 통합하는 경우 다음 내용을 추가하면 체계성이 더욱 높아진다.

* 설비관리의 5대 기능

  * 계획(Planning)
  * 운영(Operation)
  * 보전(Maintenance)
  * 개선(Improvement)
  * 폐기(Replacement)

* 설비수명주기비용(LCC, Life Cycle Cost)

* RAMS

  * Reliability
  * Availability
  * Maintainability
  * Safety

* FMEA와 설비 신뢰성 관리

* RCM(Reliability Centered Maintenance)

* ISO 55000 자산관리 체계

특히 최근 제조혁신 관점에서는 **TPM → CBM → PdM → AI 기반 자율보전**으로 발전하는 흐름을 하나의 그림으로 정리하면 전체 설비관리 체계를 이해하는 데 효과적이다.
