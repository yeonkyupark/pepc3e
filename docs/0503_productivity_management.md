## 생산성 관리

생산성 관리(Productivity Management)는 기업이 보유한 생산요소를 효율적으로 활용하여 최대의 성과를 창출하기 위한 관리활동이다. 생산성은 단순한 생산량 증가가 아니라 투입 대비 산출의 비율을 의미하며, 원가절감·품질향상·납기단축·유연성 확보를 동시에 달성하는 핵심 경영지표이다. 최근에는 스마트팩토리, AI, 디지털 트윈 기술과 결합하여 실시간 생산성 관리 체계로 발전하고 있다.

```text
생산요소(Input)
  ↓
변환(Process)
  ↓
산출(Output)
  ↓
생산성 향상
```

## 생산성의 개요

### 생산성의 정의

생산성(Productivity)이란 일정 기간 동안 투입된 생산요소 대비 산출된 결과물의 비율을 의미한다.

```text
생산성 = 산출(Output) / 투입(Input)
```

### 생산성의 중요성

* 기업 경쟁력 향상
* 원가 절감
* 수익성 증대
* 국가 경쟁력 강화
* 자원 활용 효율화
* 지속가능경영 실현

### 생산성 향상의 기본 방향

```text
산출 증가
    ↑
생산성 향상
    ↓
투입 감소
```

즉, 동일 투입으로 산출을 증가시키거나 동일 산출을 더 적은 투입으로 달성하는 것이 생산성 향상의 본질이다.

## 생산성의 유형

### 생산요소별 생산성

#### 노동생산성

노동 투입 대비 산출 수준

```text
노동생산성
=
생산량 / 노동시간
```

#### 설비생산성

설비 활용에 따른 생산효율

```text
설비생산성
=
생산량 / 설비가동시간
```

#### 자재생산성

자재 투입 대비 산출 수준

```text
자재생산성
=
산출량 / 자재투입량
```

#### 에너지 생산성

에너지 사용 효율

```text
에너지 생산성
=
생산량 / 에너지 사용량
```

### 생산성 측정 수준

| 구분                       | 내용              |
| ------------------------ | --------------- |
| 부분생산성                    | 단일 투입요소 기준      |
| 총요소생산성(TFP)              | 노동·자본 등 복수요소 고려 |
| 총생산성(Total Productivity) | 모든 투입요소 고려      |

### 생산성과 수익성의 관계

생산성과 수익성은 밀접하지만 동일 개념은 아니다.

```text
수익성
=
생산성 × 가격회수효율
```

생산성이 높아도 판매가격 하락이나 시장 악화로 수익성이 낮아질 수 있다.

## OEE와 생산성 관리

### OEE의 개요

OEE(Overall Equipment Effectiveness)는 설비의 종합효율을 측정하는 대표 지표이다.

### OEE 구성요소

```text
OEE
=
가동률
×
성능가동률
×
양품률
```

| 요소    | 의미            |
| ----- | ------------- |
| 가동률   | 계획시간 대비 실제 가동 |
| 성능가동률 | 설계속도 대비 실제 속도 |
| 양품률   | 생산량 대비 양품 비율  |

### 설비종합효율의 의의

* 설비손실 가시화
* TPM 성과측정
* 생산성 향상 기반

## 생산성 평가모델

### APC 생산성 모델

미국생산성센터(APC)는 생산성을 다음 세 요소의 결합으로 설명한다.

```text
생산성
=
효율성(Efficiency)
+
효과성(Effectiveness)
+
수익성(Profitability)
```

### Sink의 7대 성과지표

| 지표                   | 의미      |
| -------------------- | ------- |
| Effectiveness        | 효과성     |
| Efficiency           | 효율성     |
| Quality              | 품질      |
| Productivity         | 생산성     |
| Quality of Work Life | 근로생활의 질 |
| Innovation           | 혁신      |
| Profitability        | 수익성     |

기술사 시험에서는 생산성을 단순한 효율지표가 아니라 조직성과 관점에서 설명하는 경우가 많다.

## 생산성과 지속가능성

최근 생산성 관리는 단순 효율성 향상을 넘어 ESG 관점까지 확대되고 있다.

### Green Productivity

생산성과 환경성을 동시에 향상시키는 접근법이다.[^1]

#### 주요 목표

* 에너지 절감
* 탄소배출 감소
* 자원효율 향상
* 폐기물 최소화

### Carbon Productivity

```text
Carbon Productivity
=
부가가치
/
탄소배출량
```

탄소중립 시대의 새로운 생산성 지표로 활용되고 있다.

## 생산성 관리의 종합

생산성 관리는 기업의 경쟁력을 결정하는 핵심 관리활동으로, 투입 대비 산출의 효율성을 향상시키는 것을 목표로 한다. 전통적으로는 작업연구와 설비관리 중심으로 발전하였으나, 최근에는 Lean, Six Sigma, TPM, TOC 등 혁신기법과 결합하여 전사적 생산성 혁신 체계로 발전하였다. 또한 스마트팩토리, AI, Digital Twin, ESG 기반 Green Productivity가 확산되면서 생산성 관리의 범위는 효율성 중심에서 지속가능성과 기업가치 창출 중심으로 확대되고 있다.

### 범위 연계

| 연계 영역       | 주요 내용                    |
| ----------- | ------------------------ |
| 작업방법 및 시간연구 | 표준시간, 작업개선               |
| 프로세스 분석과 개선 | Lean, TOC, VSM           |
| 품질관리        | TQM, Six Sigma           |
| 설비보전관리      | TPM, OEE                 |
| 원가관리        | 원가절감과 생산성                |
| 가치공학        | 가치 향상                    |
| 스마트팩토리      | MES, AI, Digital Twin    |
| 생산혁신기술      | WCM, Lean Transformation |

### 각주

[^1]: Green Productivity는 1994년 Asian Productivity Organization에서 제안한 개념으로, 생산성 향상과 환경보호를 동시에 추구하는 지속가능 생산성 관리 체계이다.
