## 표준작업관리

표준작업관리(Standard Work Management)는 현재 시점에서 가장 안전하고 효율적이며 품질이 확보된 작업방법을 표준화하고 이를 유지·개선하는 관리활동이다. 표준작업은 생산성 향상, 품질 안정화 및 지속적 개선의 출발점이며, TPS(Toyota Production System)에서는 표준작업이 카이젠(Kaizen)의 기반으로 활용된다.

### 표준작업관리의 개요

#### 표준작업의 정의

표준작업(Standard Work)이란 정해진 품질, 수량 및 납기 조건을 만족하기 위하여 작업순서와 작업방법을 표준화한 작업기준이다.

#### 표준작업관리의 목적

* 작업방법 표준화
* 품질 안정화
* 생산성 향상
* 안전 확보
* 교육훈련 효율화
* 지속적 개선 기반 구축

#### 표준작업의 중요성

표준이 없는 개선은 일시적 성과에 그치지만, 표준화된 작업은 개선성과를 지속적으로 유지할 수 있다.

```text
개선
 ↓
표준화
 ↓
유지
 ↓
재개선
 ↓
신표준
```

## 표준작업의 구성요소

TPS에서는 표준작업을 세 가지 요소로 설명한다.[^1]

### 작업순서(Standard Work Sequence)

작업자가 수행해야 할 최적의 작업순서

### 표준시간(Standard Time)

작업을 수행하는 데 필요한 기준시간

### 표준재공량(Standard WIP)

공정을 원활하게 운영하기 위해 필요한 최소 재공품 수량

```text
표준작업

=
작업순서
+
표준시간
+
표준재공량
```

[^1]: Toyota Production System의 표준작업 3요소.

## 표준시간

### 표준시간의 정의

표준시간(Standard Time)이란 숙련된 작업자가 정상적인 작업속도로 일정한 조건에서 작업할 때 필요한 시간이다.

### 시간의 구성

```text
표준시간

=

정미시간
+
여유시간
```

### 여유시간의 종류

| 구분   | 내용         |
| ---- | ---------- |
| 인적여유 | 피로, 생리적 욕구 |
| 작업여유 | 작업조건 변화    |
| 관리여유 | 작업지연, 관리손실 |

### 표준시간의 활용

* 생산계획 수립
* 공수 산정
* 원가계산
* 설비능력 산정
* 생산성 측정

!!! note "관련 단원"

```
표준시간 측정방법은 「작업방법 및 시간연구」에서 상세히 설명한다.
```

## 작업표준서

### 작업표준서의 정의

작업표준서는 표준작업을 문서화한 기준서이다.

### 작업표준서의 구성

* 작업목적
* 작업순서
* 작업방법
* 품질기준
* 안전사항
* 사용설비 및 공구

### 작업표준서 예시

| 항목   | 내용              |
| ---- | --------------- |
| 작업명  | 조립공정            |
| 작업순서 | 부품 조립 → 체결 → 검사 |
| 표준시간 | 5분              |
| 품질기준 | 체결토크 준수         |
| 안전사항 | 보호장비 착용         |

### 작업표준서의 효과

* 작업방법 통일
* 품질 편차 감소
* 작업자 교육 효율화

## 표준작업의 작성절차

### 작성절차

```text
작업분석
    ↓
작업측정
    ↓
최적방법 선정
    ↓
표준시간 설정
    ↓
작업표준서 작성
    ↓
교육 및 적용
```

### 주요 고려사항

#### 품질

불량이 발생하지 않는 작업방법

#### 생산성

최소 자원으로 최대 성과 달성

#### 안전성

산업재해 예방

#### 작업자 수용성

현장 적용 가능성 확보

## 표준작업의 유지관리

### 유지관리의 필요성

표준은 수립보다 유지가 중요하며, 지속적인 관리가 필요하다.

### 유지관리 활동

#### 교육훈련

작업자의 표준 준수 역량 확보

#### 작업감사

표준 준수 여부 확인

#### 표준준수율 관리

표준과 실제 작업의 차이 분석

#### 정기 개정

환경 변화에 따른 표준 업데이트

### 표준준수율

```text
표준준수율

=

표준준수 작업수
─────────────────
전체 작업수
× 100
```

## 표준작업과 개선활동

### 표준화와 개선의 관계

표준화와 개선은 반복적으로 순환한다.

```text
현행표준
    ↓
개선활동
    ↓
성과확인
    ↓
표준개정
    ↓
신표준
```

### 카이젠(Kaizen)

표준을 기반으로 지속적으로 개선하는 활동

### SDCA와 PDCA

#### SDCA

표준 유지 중심 관리사이클

```text
Standard
    ↓
Do
    ↓
Check
    ↓
Act
```

#### PDCA

개선 중심 관리사이클

```text
Plan
    ↓
Do
    ↓
Check
    ↓
Act
```

### 관계

```text
SDCA
 ↓
안정화
 ↓
PDCA
 ↓
개선
 ↓
신표준
 ↓
SDCA
```

TPS는 표준작업을 모든 개선활동의 출발점으로 본다.

* 이상상태 식별
* 품질 확보

### 표준작업과 Jidoka

## 표준작업과 품질관리

### 표준화의 역할

품질은 검사보다 표준화된 작업에서 확보된다.

### 품질관리와의 관계

| 품질관리    | 표준작업    |
| ------- | ------- |
| 품질기준 설정 | 작업기준 설정 |
| 품질측정    | 작업측정    |
| 이상관리    | 표준이탈 관리 |
| 개선활동    | 표준개정    |

### 예방품질

표준작업은 불량을 사전에 예방하는 예방품질 활동의 기반이 된다.

## 시각적 관리(Visual Management)

### 개요

작업상태를 누구나 쉽게 이해할 수 있도록 시각적으로 표현하는 관리기법

### 주요 기법

#### 안돈(Andon)

이상 발생 시 시각적으로 알리는 시스템

#### 작업표준 게시

현장 작업기준의 시각화

#### 색상관리

자재, 설비 및 작업구역 구분

#### 생산관리판

생산실적과 목표의 실시간 표시

### 기대효과

* 이상 조기 발견
* 의사소통 향상
* 현장관리 강화

## 표준작업관리의 종합

표준작업관리는 생산성과 품질을 안정적으로 유지하기 위한 생산관리의 핵심 기반이다. 작업순서, 표준시간 및 표준재공량을 체계적으로 관리함으로써 작업의 일관성을 확보하고 품질 편차를 최소화할 수 있다. 또한 표준화는 개선활동의 출발점이며, TPS의 카이젠과 스마트팩토리의 디지털 작업관리 역시 표준작업을 기반으로 운영된다.

### 범위 연계

| 연계 영역       | 주요 내용               |
| ----------- | ------------------- |
| 작업계획수립      | 작업표준, 공수계획          |
| 작업방법 및 시간연구 | 표준시간 설정             |
| 공정개선        | ECRS, 카이젠           |
| 품질관리        | 예방품질, 작업표준화         |
| 설비보전관리      | 점검표준, 작업표준          |
| 스마트팩토리      | MES, 전자표준서, AR 작업지원 |

### 보완 권장 사항

기존 초안에는 표준작업의 개념과 작성 절차는 잘 포함되어 있으나, 기술사 및 생산관리 체계 관점에서는 다음 내용을 추가하면 완성도가 높아진다.

* 표준(Standard), 규격(Specification), 기준(Criterion)의 차이
* 표준화(Standardization)의 원칙(단순화·전문화·통일화)
* 작업표준서, 검사표준서, 설비점검표준서 비교
* 표준작업조합표(Standard Work Combination Sheet)
* 표준작업지도서(Standard Work Sheet)
* 표준작업과 택트타임의 관계
* 표준작업과 라인 밸런싱의 관계

특히 TPS에서 중요하게 다루는 **표준작업조합표, 표준작업지도서, 표준작업 3요소**는 별도 소단원으로 구성하는 것을 권장한다.
