## 경쟁우위 생산전략

기업은 시장에서 경쟁우위를 확보하기 위해 경쟁전략을 수립하고, 이를 생산활동으로 구체화하기 위해 생산전략을 수립한다. 생산전략은 품질(Q), 원가(C), 납기(D), 유연성(F), 서비스(S)와 같은 경쟁우선순위를 설정하고 이를 실현할 수 있는 생산시스템을 구축하는 활동이다.

생산시스템은 산업혁명 이후 생산성 향상을 목표로 발전해 왔으며, 테일러의 과학적 관리법, 포드의 대량생산체계, 도요타 생산시스템(TPS)을 거쳐 오늘날 스마트팩토리와 지속가능 생산체계로 진화하고 있다.

### 생산전략의 개요

#### 생산전략의 정의

생산전략(Operations Strategy)이란 기업의 경쟁전략을 생산기능 측면에서 구체화하여 생산자원(인력, 설비, 자재, 기술, 정보)을 장기적·체계적으로 배분하고 운영하는 전략이다.[^1]

#### 생산전략의 목적

* 경쟁우위 확보
* 고객가치 창출
* 생산성 향상
* 기업성과 극대화
* 지속가능한 성장기반 구축

#### 전략의 위계

```text
기업전략(Corporate Strategy)
      ↓
사업전략(Business Strategy)
      ↓
생산전략(Operations Strategy)
      ↓
생산시스템 및 운영관리
```

| 구분   | 핵심질문        | 주요 의사결정       |
| ---- | ----------- | ------------- |
| 기업전략 | 무엇을 할 것인가   | 사업영역, 사업포트폴리오 |
| 사업전략 | 어떻게 경쟁할 것인가 | 경쟁방식, 시장선택    |
| 생산전략 | 어떻게 생산할 것인가 | 생산역량 구축       |

[^1]: Wickham Skinner는 생산을 단순 지원기능이 아닌 경쟁우위 창출의 핵심 기능으로 정의하며 생산전략 개념을 체계화하였다.

### 경쟁전략과 생산전략

#### 포터의 본원적 경쟁전략

포터는 기업이 경쟁우위를 확보하기 위해 원가우위(Cost Leadership) 또는 차별화(Differentiation)를 선택해야 한다고 주장하였다.[^2]

```text
                경쟁범위

          광범위         집중
      ┌─────────┬─────────┐
원가  │ 원가우위 │ 원가집중 │
      ├─────────┼─────────┤
차별화│ 차별화   │ 차별화집중│
      └─────────┴─────────┘
```

#### 경쟁전략과 생산전략의 연계

| 경쟁전략 | 생산전략            |
| ---- | --------------- |
| 원가우위 | 자동화, 표준화, 대량생산  |
| 차별화  | 고품질 생산, 첨단기술 활용 |
| 집중화  | 유연생산, 고객맞춤 생산   |

#### Stuck in the Middle

원가우위와 차별화를 동시에 추구하여 어느 경쟁우위도 확보하지 못한 상태

* 수익성 저하
* 시장지위 약화
* 경쟁력 상실

[^2]: Michael E. Porter, *Competitive Strategy*, 1980.

### 경쟁우선순위(QCDFS)

#### 의의

경쟁전략을 생산활동으로 구현하기 위해 설정하는 핵심 성과지표

#### 경쟁우선순위

| 구분          | 의미  |
| ----------- | --- |
| Quality     | 품질  |
| Cost        | 원가  |
| Delivery    | 납기  |
| Flexibility | 유연성 |
| Service     | 서비스 |

#### Order Qualifier와 Order Winner

##### Order Qualifier

시장 진입을 위해 반드시 충족해야 하는 최소 요구조건

##### Order Winner

고객의 최종 구매를 결정하는 차별화 요소

| 산업   | Qualifier | Winner  |
| ---- | --------- | ------- |
| 자동차  | 품질        | 가격, 디자인 |
| 반도체  | 품질        | 납기      |
| 스마트폰 | 품질        | 성능, 생태계 |

#### Trade-off 관계

전통적 생산관리에서는 경쟁요소 간 상충관계가 존재한다고 보았다.

| 경쟁요소      | 상충관계           |
| --------- | -------------- |
| 품질 ↔ 원가   | 품질향상 비용 증가     |
| 유연성 ↔ 생산성 | 다품종 생산 시 효율 저하 |
| 납기 ↔ 설비효율 | 여유설비 확보 필요     |
| 서비스 ↔ 원가  | 고객대응 비용 증가     |

#### Sand Cone Model

Ferdows와 De Meyer는 경쟁우위가 상충관계보다 역량축적을 통해 달성된다고 주장하였다.[^3]

```text
품질
 ↓
납기
 ↓
유연성
 ↓
원가
```

품질역량을 기반으로 납기, 유연성, 원가경쟁력이 순차적으로 축적된다.

[^3]: Ferdows & De Meyer, *Lasting Improvements in Manufacturing Performance*, 1990.

### 생산전략의 의사결정 영역

생산전략은 생산시스템 설계를 위한 구조적 의사결정과 운영 효율화를 위한 인프라적 의사결정으로 구성된다.

#### 구조적 의사결정

* 생산능력(Capacity)
* 입지(Location)
* 설비배치(Layout)
* 공정(Process)
* 기술(Technology)

#### 인프라적 의사결정

* 품질관리
* 생산계획
* 재고관리
* 인력관리
* 조직 및 성과관리

#### 생산전략과 출제범위 연계

| 생산전략  | 관련 단원    |
| ----- | -------- |
| 공정전략  | 공정설계     |
| 설비전략  | 설비관리     |
| 품질전략  | 품질관리     |
| SCM전략 | 공급망관리    |
| 인력전략  | 인력과 직무설계 |

## 생산시스템의 발전과 생산철학

### 생산시스템의 개요

생산전략이 방향을 제시한다면 생산시스템은 이를 실제 운영으로 구현하는 수단이다.

#### 정의

투입요소를 고객가치가 있는 산출물로 전환하는 유기적 시스템

#### 구성요소

```text
투입(Input)
 ↓
변환(Transformation)
 ↓
산출(Output)

↑
피드백(Feedback)

+ 환경(Environment)
```

#### 생산시스템의 목표

* 품질 향상
* 원가 절감
* 납기 단축
* 유연성 확보

즉 QCDFS 달성이 생산시스템의 궁극적 목표이다.

### 생산시스템 발전과정

생산시스템의 역사는 "효율성 → 원가 → 품질 → 유연성 → 지능화"의 진화 과정으로 볼 수 있다.

```text
수공업 생산

↓

Taylor
(생산성)

↓

Ford
(원가)

↓

Toyota
(품질·유연성)

↓

Lean

↓

Smart Factory
(지능화)
```

| 시대        | 핵심가치    | 대표시스템         |
| --------- | ------- | ------------- |
| 1900년대 초  | 생산성     | Taylor System |
| 1900년대 중반 | 원가      | Ford System   |
| 1970년대 이후 | 품질·유연성  | TPS           |
| 1990년대 이후 | 낭비제거    | Lean          |
| 현재        | 지능화·최적화 | Smart Factory |

### 테일러 시스템

#### 개요

프레더릭 테일러(F. W. Taylor)가 제안한 과학적 관리법(Scientific Management)[^4]

#### 핵심원리

* 작업의 표준화
* 작업방법의 과학화
* 계획과 실행의 분리
* 성과중심 보상

#### 주요 제도

* 기획부제
* 직능별 직장제
* 지도표제
* 차별적 성과급제

#### 의의

현대 산업공학(IE)과 작업연구의 출발점

[^4]: Frederick W. Taylor, *The Principles of Scientific Management*, 1911.

### 포드 시스템

#### 개요

헨리 포드가 구축한 이동조립라인 기반 대량생산체계

#### 핵심요소

* 표준화(Standardization)
* 전문화(Specialization)
* 단순화(Simplification)
* 조립라인(Assembly Line)

#### 의의

규모의 경제를 통한 원가우위 실현

#### 한계

* 제품 다양성 부족
* 고객맞춤 생산 한계
* 대량재고 발생

### 도요타 생산시스템(TPS)

#### 개요

오노 다이이치와 신이고가 발전시킨 린 생산철학으로 낭비 제거를 통해 고객가치를 극대화하는 생산시스템이다.[^5]

#### TPS의 궁극적 목표

* 원가 절감
* 품질 향상
* 리드타임 단축
* 유연성 확보

#### TPS의 기본구조

```text
목표 : 원가절감

       ▲
   JIT + Jidoka

       ▲
표준작업 · 평준화 · 카이젠
```

#### TPS의 두 기둥

| 구분     | 의미                       |
| ------ | ------------------------ |
| JIT    | 필요한 것을 필요한 때 필요한 양만 생산   |
| Jidoka | 이상 발생 시 즉시 정지하여 불량 유출 방지 |

#### 7대 낭비(Muda)

* 과잉생산
* 대기
* 운반
* 가공
* 재고
* 동작
* 불량

#### 가동률과 가동률

| 구분  | 의미                |
| --- | ----------------- |
| 稼動率 | 실제 생산활동 비율        |
| 可動率 | 설비가 언제든 가동 가능한 상태 |

#### 현대적 의의

TPS는 Lean Manufacturing의 기반이 되었으며, 현대 생산전략에서 QCDF를 동시에 달성하기 위한 대표적 운영모델로 평가된다.

[^5]: Taiichi Ohno, *Toyota Production System*, 1978.

### 생산전략의 최근 동향

시장 불확실성과 디지털 전환이 가속화되면서 생산전략 역시 새로운 방향으로 발전하고 있다.

#### Lean Manufacturing

* 낭비 제거
* 가치흐름 중심
* 지속적 개선

#### Agile Manufacturing

* 시장변화 신속 대응
* 고객맞춤 생산
* 고유연성 확보

#### Mass Customization

* 대량생산 효율성
* 맞춤생산 차별화

#### Smart Factory

* CPS 기반 자율운영
* AI 기반 의사결정
* Digital Twin 활용

#### ESG 생산전략

* 탄소중립
* 에너지 효율화
* 자원순환
* 지속가능 공급망 구축

!!! note "관련 단원"

```
Smart Factory는 「정보화활용기술」, Lean 및 Agile은 「생산혁신기술」에서 상세히 설명한다.
```
